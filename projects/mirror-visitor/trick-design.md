# 诡计设计 — 《镜中访客》

## 一、核心诡计概述

- **谜面**：叶知秋的来访者接连遭受神秘人骚扰，有人在利用咨询中获取的隐私信息伤害他们。叶知秋展开调查，追踪这个"镜中访客"。
- **谜底**："镜中访客"就是叶知秋本人在解离状态下的行为。她在失去意识的时段内接近来访者，利用咨询中获得的信息对他们实施心理操控。
- **诡计机制**：第一人称叙事 + 自我欺骗型不可靠叙述。叶知秋的叙述是"真实的"——她真的不知道自己做了什么。她描述的"空白时间"被她自己填充为"我睡着了"或"我在发呆"。读者随叶知秋的视角移动，自然跳过了这些空白。

## 二、公平性验证

**不可靠叙述者的公平性协议**（参照 unreliable-narrator.md）：

本作采用"自我欺骗型"不可靠叙述——公平性标准为"中等"。

- [x] 叙述者的限制在早期被建立（第1章就展示了失眠和时间感模糊）
- [x] 至少一个其他角色对同一事件有不同描述（苏远的视角）
- [x] 叙述者自己偶尔表达对自己记忆的不确定（"我好像..."、"应该是..."）
- [x] Knox #7 变体：叙述者即犯人——但本作不是隐瞒（她真的不知道）
- [x] 每句话字面上都是真的（叶知秋没有说谎，只是她的认知是扭曲的）

## 四、诡计可行性分析

- **物理可行性**：解离性身份障碍（DID）在临床上有据可查。解离状态下的行为不被主人格记忆，符合医学事实。
- **心理可行性**：叶知秋的童年创伤→过度代偿的"治愈者"身份→压力下解离，这条因果链在心理学上成立。
- **信息控制可行性**：第一人称叙事天然限制视角。叶知秋"空白时间"的叙述跳过是自然的——"我不记得"就是不写。读者不会注意到缺失的时间，就像叶知秋自己不注意一样。
