# 镜头语法完全手册

> 景别、角度、运动、构图——镜头语言的完整词汇表与语法规则
> 每个镜头选择都是一个叙事决策，不存在"中性"的镜头

---

## 一、景别体系（Shot Size）

景别是导演最基本的情感控制工具。核心法则：**情感距离 = 物理距离**。

### 完整景别谱系

```
远 ←───────────────────────────────────────→ 近

EWS   ELS   VLS   LS    MLS   FS    MS    MCS   MCU   CU    BCU   ECU
极宽  大远  远景  全景  中全  满景  中景  中近  中特  特写  大特  极特
```

#### 1. 极宽景 / 超大远景 (Extreme Wide Shot, EWS)

**画面内容**: 巨大的自然或城市景观，人物几乎不可见
**情感效果**: 宇宙感、渺小、命运、史诗开场
**叙事用途**: 确立故事世界的规模和调性
**悬疑用途**: 建立故事发生的封闭/开放空间感

```
适用场景:
  - 电影开篇的第一个镜头
  - "故事发生在xxx"的空间宣告
  - 孤独感的极端表达
  - 结局的释放/超越

技术参数:
  - 通常使用广角镜头（14-35mm）
  - 时长 3-8 秒
  - 常配合航拍/无人机
```

#### 2. 大远景 (Extreme Long Shot, ELS)

**画面内容**: 环境为主，人物可识别但很小
**情感效果**: 环境主导、人物与世界的关系
**叙事用途**: 建立地理关系、展示旅程

```
悬疑专用技巧:
  - 在大远景中隐藏一个微小异常（远处的人影/窗口的灯光）
  - 观众可能注意到也可能错过 → Fair-play线索植入
  - 重看时才发现"凶手一直在画面里"
```

#### 3. 远景 (Long Shot, LS)

**画面内容**: 人物全身可见，环境信息丰富
**情感效果**: 客观观察、社交距离、环境压力
**叙事用途**: 角色进入/离开空间、展示身体语言

```
悬疑专用技巧:
  - 角色在远景中的走路姿态可以作为身份识别线索
  - 远景中的"不经意"扫过某个关键场景元素
```

#### 4. 全景 (Full Shot, FS)

**画面内容**: 人物头到脚完整呈现
**情感效果**: 完整的人物印象、行动自由度
**叙事用途**: 角色初次登场、肢体动作表演

#### 5. 中全景 (Medium Long Shot, MLS)

**画面内容**: 膝盖以上
**情感效果**: 社交场合的典型距离
**叙事用途**: 群戏、走动中的对话

#### 6. 中景 (Medium Shot, MS)

**画面内容**: 腰部以上
**情感效果**: 对话的自然距离、中性观察
**叙事用途**: 对话场景的工作马（最常用景别）

```
警告: 过度使用中景是分镜初学者最常见的错误。
中景是"安全"的选择，但也是最无力的选择。
每个中景都应该问："为什么不是特写或全景？"
```

#### 7. 中近景 (Medium Close-Up, MCU)

**画面内容**: 胸部以上
**情感效果**: 亲密对话、注意力集中在面部
**叙事用途**: 采访、正式对话、新闻播报

#### 8. 特写 (Close-Up, CU)

**画面内容**: 面部占满画面
**情感效果**: 情感暴露、亲密、无处可逃
**叙事用途**: 关键反应、情感高潮、谎言识别

```
悬疑专用技巧:
  - 特写是"测谎仪"——角色在特写中说谎，
    观众潜意识会感到不适（微表情）
  - 在揭示真相前的最后一个镜头用特写:
    犯人/受害者的面部 → 让观众记住这张脸
```

#### 9. 大特写 (Big Close-Up, BCU)

**画面内容**: 面部的一部分（眼睛+鼻子 或 嘴唇）
**情感效果**: 极端情感、无法掩饰的真相
**叙事用途**: 恐惧、决心、微表情捕捉

#### 10. 极特写 (Extreme Close-Up, ECU)

**画面内容**: 眼睛、嘴唇、手指、物件细节
**情感效果**: 极端聚焦、时间暂停感
**叙事用途**: 关键线索揭示、情感巅峰

```
悬疑专用技巧:
  - ECU 用于线索回溯蒙太奇——
    快速闪回之前出现过的物件细节
  - 手的ECU：拿钥匙/藏东西/手指颤抖
  - 眼睛的ECU：识别/恐惧/决心的瞬间
```

#### 11. 插入镜头 (Insert Shot)

**画面内容**: 独立的物件/细节镜头
**情感效果**: 注意力强制引导、信息传递
**叙事用途**: 展示关键物品、时间/日期、文字

```
悬疑专用技巧:
  - 公平线索的核心载体
  - 第一次出现：快速INSERT，不给观众太多思考时间
  - 揭示时再次出现：延长时长，配合音效
```

### 景别选择决策树

```
这个镜头要传达什么？
├── 环境/空间关系 → ELS / LS
├── 完整的人物行动 → FS / MLS
├── 对话/社交互动 → MS / MCU
├── 情感反应/内心 → CU / BCU
├── 极端情感/真相 → ECU
├── 关键物件/线索 → INSERT
└── 不确定 → 问自己："此刻故事的重心是什么？"
    ├── 环境压迫人 → 宽景（人小环境大）
    ├── 人抗争环境 → 窄景（人大环境小）
    └── 内心冲突 → 特写（只有脸）
```

## 二、角度体系（Camera Angle）

角度是权力关系的视觉编码。核心法则：**摄影机的高度 = 观众的心理位置**。

### 完整角度谱系

#### 1. 平视 (Eye Level)

```
心理效果: 平等、客观、正常
何时用: 默认选择。中性叙述时使用。
何时不用: 需要表达权力差异或心理异常时。
```

#### 2. 高角度 / 俯拍 (High Angle)

```
心理效果: 脆弱、被审视、渺小、被控制
何时用:
  - 受害者视角
  - 角色处于劣势
  - 审讯中的被审者
  - 角色做出屈服的决定
变体:
  - 轻微俯拍（15-30°）: 轻微劣势
  - 明显俯拍（30-60°）: 明显弱势
  - 极端俯拍（60-80°）: 被完全压制
```

#### 3. 低角度 / 仰拍 (Low Angle)

```
心理效果: 权威、威胁、崇高、压迫
何时用:
  - 反派登场/展示力量
  - 权力人物的说话镜头
  - 角色做出强势决定
  - 建筑物的压迫感
变体:
  - 轻微仰拍: 自信
  - 明显仰拍: 权威
  - 极端仰拍: 恐惧/崇拜
```

#### 4. 鸟瞰 (Bird's Eye View / Top Shot)

```
心理效果: 全知、命运感、脱离人性
何时用:
  - 犯罪现场全貌
  - 角色被困的空间
  - "上帝视角"揭示真相
  - 舞蹈/仪式/群体行为的几何美感
悬疑用法:
  - 从鸟瞰展示案发现场的空间关系——
    观众获得侦探没有的信息
```

#### 5. 虫视 (Worm's Eye View)

```
心理效果: 极端弱势、异化、怪异
何时用:
  - 角色倒在地上
  - 恐惧到无法站立
  - 非人视角（动物/儿童）
  - 表达世界的扭曲感
```

#### 6. 荷兰角 / 倾斜角 (Dutch Angle / Canted Angle)

```
心理效果: 不安、疯狂、世界失衡
何时用:
  - 角色心理失衡
  - 现实被扭曲的时刻
  - 药物/酒精/精神崩溃
  - 恐怖/惊悚的氛围强化
注意:
  - 5-15°: 微妙不安
  - 15-30°: 明显失衡
  - 30-45°: 极端疯狂
  - 过度使用会削弱效果——每场戏最多2-3个
```

#### 7. 过肩镜头 (Over-the-Shoulder, OTS)

```
心理效果: 对话中的空间感、关注说话者
何时用:
  - 两人对话的标准覆盖
  - 审讯场景
  - 建立两人的空间关系
变体:
  - Dirty OTS: 前景人物占画面更多（压迫感）
  - Clean OTS: 前景人物最小化（聚焦说话者）
  - Single: 完全没有前景人物（孤立感）
```

#### 8. 主观镜头 (Point of View, POV)

```
心理效果: 完全代入、第一人称体验
何时用:
  - 角色发现线索
  - 恐惧中看到威胁
  - 通过角色的眼睛重新审视已知信息
悬疑专用:
  - POV + 不可靠叙事者 = 观众和角色一起被骗
  - 侦探的POV看向嫌疑人 → 观众参与推理
  - 凶手的POV（隐藏身份）→ 经典悬疑手法
```

## 三、摄影机运动体系（Camera Movement）

运动是能量和节奏的物理载体。核心法则：**摄影机的运动 = 叙事的能量流**。

### 运动语义速查

| 运动 | 情感语义 | 最佳搭配 |
|------|---------|---------|
| 静止 | 观察、等待、压力、客观 | 审讯、对峙、凝视 |
| 推近 | 发现、聚焦、接近真相 | 揭示、顿悟、下定决心 |
| 拉远 | 疏远、揭示全貌、释放 | 结局、真相大白、孤独 |
| 横移 | 跟随、平行、引导视线 | 行走、追踪、环境展示 |
| 升起 | 解放、超越、离开 | 结局、死亡、升华 |
| 下降 | 进入、压迫、聚焦 | 开场、进入封闭空间 |
| 环绕 | 审视、困惑、多维 | 审讯、内心冲突、犹豫 |
| 手持 | 不稳定、真实、焦虑 | 追逐、恐慌、纪实 |
| 稳定器 | 流畅、超自然、幽灵 | 跟踪、闪灵式走廊 |

### Dolly Zoom（眩晕变焦）

Hitchcock 在《Vertigo》中首创的技法：

```
原理: 摄影机后拉 + 镜头变焦推近（或反向）
效果: 背景变大/变小但主体大小不变 → 空间扭曲感
情感: 顿悟、恐惧、认知失调、世界观崩塌

适用场景:
  - 角色发现重大真相的瞬间
  - 恐高/恐惧的主观体验
  - "一切都变了"的认知转折点

使用频率: 一部电影最多2-3次，每次都应该是叙事转折点
```

### 长镜头 vs 快切

```
长镜头（>30s）的叙事效果:
  - 真实感：不经剪辑的连续性 = 可信度
  - 张力累积：无法通过切出来释放压力
  - 空间完整性：观众理解空间关系
  - 表演空间：给演员完整的情感弧线

快切（<2s/shot）的叙事效果:
  - 加速：节奏加快，紧迫感
  - 碎片化：混乱、恐慌、信息过载
  - 主观体验：角色的心理碎裂
  - 能量释放：高潮时刻的能量爆发
```

## 四、构图原则（Composition）

### 核心构图法则

#### 三分法（Rule of Thirds）

```
将画面分为3×3的网格：
┌─────┬─────┬─────┐
│     │     │     │
│   ●─┤─────┤─●   │  ● = 力量点（视觉重心）
│     │     │     │
├─────┼─────┼─────┤
│     │     │     │
│   ●─┤─────┤─●   │
│     │     │     │
└─────┴─────┴─────┘

关键元素放在力量点或线上：
  - 人物眼睛在上1/3线
  - 重要物件在力量点
  - 地平线在上或下1/3线
```

#### 引导线（Leading Lines）

```
利用场景中的线条引导视线到焦点：
  - 走廊/道路 → 消失点处的人物
  - 桌子边缘/栏杆 → 关键物品
  - 光线方向 → 重要区域
  - 人物视线 → 下一个焦点

悬疑用法：
  引导线指向线索位置，但观众可能被主要人物吸引
  → 线索"在画面中"但不被注意 → 公平线索
```

#### 框中框（Frame within Frame）

```
门框/窗框/镜子/屏幕 作为画中画：
  - 门框内的人物 → 被困、受限
  - 窗户中的倒影 → 双重性、秘密
  - 镜子 → 自我审视、分裂人格
  - 屏幕中屏幕 → 监控、被观察

悬疑用法：
  镜中的反射与实际不同 → 不可靠叙事的视觉暗示
  窗户中看到不应该在那里的人影 → 悬念建立
```

#### 对称与破对称

```
对称构图:
  - 秩序、控制、权力、压迫
  - Kubrick 标志性技法
  - 用于建立"正常"状态

破对称:
  - 在对称构图中引入一个不对称元素
  - 暗示秩序即将被打破
  - 悬疑中：完美画面中的一个异常 = 线索
```

#### 负空间（Negative Space）

```
画面中的"空白"区域：
  - 大量负空间 → 孤独、脆弱、暴露
  - 角色面向的方向留空 → 正常（looking room）
  - 角色背后留空 → 不安（什么在身后？）
  - 头顶留大量空间 → 渺小、被压迫

悬疑用法：
  负空间是"威胁可能出现的区域"
  观众本能地注视空白处 → 恐惧的来源
```

## 五、景别转换规则

### 允许的跳级

```
平滑过渡（无需额外元素）:
  ELS → LS → FS → MS → MCU → CU → ECU
  每次跳一级是最安全的

可接受的跳级（需要理由）:
  ELS → MS : 跳过建立，直接进入（快节奏）
  MS → ECU : 突然聚焦（惊吓/发现）
  CU → ELS : 释放+建立（大呼吸）

禁忌跳级（除非有意制造不适）:
  ECU → ELS : 极端跳级，观众定向丧失
  ELS → ECU : 观众无法定位ECU的空间位置
```

### 连贯性编辑的铁律

```
1. 180度轴线规则
   两人对话时，摄影机始终在连接两人的假想线的同一侧
   越轴 → 左右关系颠倒 → 观众混淆
   例外：故意越轴表达混乱/权力反转

2. 30度规则
   相邻镜头的角度变化至少30度
   否则 → 跳切感（除非故意使用跳切）

3. 视线匹配
   A看向画面右方 → 下一镜头B从画面左方看
   不匹配 → 观众不认为他们在看彼此

4. 动作匹配
   在动作进行中切（不是完成后）
   例：手伸向门把手（切）→ 门打开
   不在动作完成后切 → 流畅感

5. 银幕方向
   角色向右走 → 跨镜头保持向右
   突然反向 → 暗示故事方向改变
```
