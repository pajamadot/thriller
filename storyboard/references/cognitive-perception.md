# 认知知觉系统 (Cognitive Perception System)

> 分镜设计的神经科学基础——注意力预测、认知负荷限制、情感启动、注意力同步
> 8 项知觉来源 + 形式化 IF/THEN 决策规则 + 可测试预测
> Band 3 (R021-R030) 认知与知觉深化产物

---

## 零、总论：为什么需要认知规则？

传统分镜设计依赖"直觉"和"经验法则"。认知知觉系统将这些直觉转化为有神经科学实验依据的形式化规则，使得：

1. **可审计**：每条规则可追溯到具体的实验发现
2. **可预测**：给定一个镜头，可预测观众的注意力落点、认知负荷、情感状态
3. **可测试**：预测可以与实际观众反应对比验证
4. **可教学**：LLM 可以根据明确的 IF/THEN 规则生成更准确的分镜

## 一、文献基础 (8+ Sources)

### Source 1: Posner (1980) — 注意力聚光灯

```
来源: Posner, M. I. (1980). "Orienting of attention."
      Quarterly Journal of Experimental Psychology, 32(1), 3-25.

核心发现:
  视觉注意力像一个"聚光灯"，可以独立于眼球运动而移动。
  两种注意力驱动模式:
    外源性 (Exogenous): 由外部刺激自动触发（突然运动、闪光、
      高对比），速度快（~100ms），不可抑制
    内源性 (Endogenous): 由内部意图驱动（"我要看那里"），
      速度慢（~300ms），可被覆盖

分镜应用:
  外源性注意力 → 决定观众在新镜头中"第一眼看到什么"
  内源性注意力 → 决定观众在镜头持续期间"选择看什么"
  设计者可以通过控制外源性线索来引导第一眼，
  然后通过构图引导内源性注意力的扫描路径
```

### Source 2: Sweller (1988) — 认知负荷理论

```
来源: Sweller, J. (1988). "Cognitive load during problem solving:
      Effects on learning." Cognitive Science, 12(2), 257-285.

核心发现:
  工作记忆容量有限。认知负荷分三类:
    内在负荷 (Intrinsic): 信息本身的复杂度（不可降低）
    外在负荷 (Extraneous): 信息呈现方式造成的额外负荷（应消除）
    关联负荷 (Germane): 将信息整合到已有图式的负荷（应促进）

  Miller (1956) 的 7+/-2 法则是上限参照:
    工作记忆一次处理 5-9 个信息组块

分镜应用:
  每个镜头 = 一次信息呈现
  景别越窄 → 画面信息越少 → 允许的信息密度越高（集中注意力）
  景别越宽 → 画面信息越多 → 需要限制叙事信息密度（分散注意力）
  外在负荷在分镜中 = 无理由的切、混乱的空间关系、不一致的轴线
  → 这些消耗认知资源但不服务叙事
```

### Source 3: Cutting, DeLong & Nothelfer (2010) — 好莱坞镜头时长的 1/f 模式

```
来源: Cutting, J. E., DeLong, J. E., & Nothelfer, C. E. (2010).
      "Attention and the evolution of Hollywood film."
      Psychological Science, 21(3), 432-439.

核心发现:
  分析 150 部好莱坞电影 (1935-2010) 的镜头时长序列。
  成功影片的镜头时长分布遵循 1/f 噪声模式 (pink noise):
    不是随机的 (白噪声/无模式)
    不是完全规律的 (棕噪声/过度可预测)
    而是介于两者之间的分形模式

  1/f 噪声的特征:
    短期内有变化 → 不无聊
    长期内有自相似的模式 → 不混乱
    人类注意力本身也呈 1/f 波动 → 匹配 = 最佳参与度

  现代电影 (2000+) 比经典电影 (1930-1960) 更接近 1/f
  → 导演无意识地进化向了认知最优模式

分镜应用:
  不要使所有镜头等长（白噪声 → 无聊）
  不要使镜头时长完全随机（高频白噪声 → 混乱）
  不要使镜头时长过于规律（棕噪声 → 可预测）
  目标: 建立节奏模式，然后在模式上做有意义的变化
  实践检验: 计算镜头时长序列的功率谱，斜率应接近 -1
```

### Source 4: Smith & Henderson (2008) — 电影观看中的注意力同步

```
来源: Smith, T. J., & Henderson, J. M. (2008).
      "Edit Blindness: The relationship between attention and
      global change blindness in dynamic scenes."
      Journal of Eye Movement Research, 2(2), 1-17.

      及 Smith, T. J. (2013). "Watching you watch movies:
      Using eye tracking to inform cognitive film theory."
      In Shimamura (Ed.), Psychocinematics.

核心发现:
  使用眼动追踪研究多名观众同时观看同一电影片段:
    在某些镜头中，所有观众的注视点高度集中 (高同步)
    在另一些镜头中，观众的注视点分散 (低同步)

  高同步触发条件:
    1. 面部特写 (最强同步诱导器)
    2. 突然运动 (外源性注意力捕获)
    3. 单一突出对象 (pop-out效应)
    4. 音频线索指向画面某区域

  低同步条件:
    1. 宽景中有多个等重要元素
    2. 静态场景无显著视觉锚点
    3. 景深均匀 (深焦/全焦)

  关键发现: 高同步时观众对剪辑的感知降低（"编辑盲"）
  → 在注意力高度集中的时刻剪切，观众不会注意到切

分镜应用:
  需要隐形的切 → 在高同步时刻切 (面部、突然运动)
  需要观众注意到切 → 在低同步时刻切 (宽景、静态)
  线索隐藏 → 在低同步时将线索放在观众注视点之外
  悬疑利用 → 高同步(面部)锁住观众注意力，
             同时在背景中安排只有重看才能发现的信息
```

### Source 5: LeDoux (1996) — 情感的双通道处理

```
来源: LeDoux, J. E. (1996). The Emotional Brain: The Mysterious
      Underpinnings of Emotional Life. Simon & Schuster.

核心发现:
  情感处理有两条并行神经通路:
    快通路 (Low Road): 感觉丘脑 → 杏仁核 (12ms)
      粗糙、快速、自动
      先于意识判断产生情感反应
      进化适应: 对威胁的快速反应 > 准确判断

    慢通路 (High Road): 感觉丘脑 → 皮层 → 杏仁核 (30-40ms)
      精细、慢速、有意识
      可以覆盖快通路的反应
      但需要更多时间和认知资源

  情感启动 (Emotional Priming):
    先呈现的情感刺激会影响后续刺激的解读
    启动效应需要最短呈现时间:
      快通路启动: ~100-200ms (阈下知觉即可)
      慢通路启动: ~500ms-1.5s (需要有意识加工)
      深度情感启动: 3-5s (需要情境建构)

分镜应用:
  闪现帧 (<0.3s) 可以通过快通路产生不安感但不被意识到
  对比切 (contrast-cut) 需要前序镜头至少 1.5-3s 建立基调
  深度反转 (如从安全到恐惧) 需要 5s+ 的基调建立
  悬疑中的"不安感"可以通过快通路阈下刺激制造:
    角色不注意的背景运动、极短的异常帧、声音微变
```

### Source 6: Hasson et al. (2008) — 电影观看中的脑间皮层同步

```
来源: Hasson, U., Landesman, O., Knappmeyer, B., Vallines, I.,
      Rubin, N., & Heeger, D. J. (2008). "Neurocinematics:
      The Neuroscience of Film." Projections, 2(1), 1-26.

核心发现:
  使用 fMRI 测量多名观众观看同一电影时的脑活动:
    某些电影片段引发高度一致的脑活动模式 (高ISC)
    另一些片段引发不一致的脑活动 (低ISC)

  ISC 最高的导演/影片:
    Hitchcock (Rope/Bang! You're Dead): 65% 皮层面积同步
    Sergio Leone (The Good, the Bad and the Ugly): 45%
    Curb Your Enthusiasm: 18%
    未剪辑的 Washington Square Park 实景: 5%

  Hitchcock 的高 ISC 来自:
    精确的注意力控制 (每个镜头只有一个注意力焦点)
    时间控制 (精确的信息释放节奏)
    情感操控 (通过编辑精确控制情感节拍)

  高 ISC 的条件:
    限制性构图 (少元素、强焦点)
    紧密的编辑节奏 (每个切都有明确目的)
    明确的因果叙事 (不含糊)

分镜应用:
  悬疑片应追求高 ISC → 精确控制每个观众在每个时刻的注意力
  方法: 限制每个镜头的注意力焦点为 1-2 个
  构图: 使用浅景深、光线引导、运动引导来强制注意力焦点
  编辑: 确保每个切都推动因果链 (= shot-logic.md 的 PROVIDES 规则)
  反面: 故意降低 ISC (宽景多元素) 用于隐藏线索
```

### Source 7: Shimamura (2013) — 心理电影学

```
来源: Shimamura, A. P. (2013). "Psychocinematics: Issues and
      Directions." In Shimamura (Ed.), Psychocinematics:
      Exploring Cognition at the Movies. Oxford University Press.

核心发现:
  提出 I-S-K-E 框架:
    I (Identify): 感知基本视觉特征 (颜色、形状、运动)
    S (Structure): 组织为有意义的对象和场景
    K (Knowledge): 与已有知识匹配 (图式、文体期望)
    E (Execute): 执行认知操作 (推理、预测、情感反应)

  电影编辑的认知成本:
    简单切 (同场景、匹配轴线): I + S (低成本)
    中等切 (跨景别、不同角度): I + S + K (中等成本)
    复杂切 (跨场景、跳跃式): I + S + K + E (高成本)

  记忆与注意力的交互:
    观众不记住每个镜头 → 记住"事件"(event)
    事件边界 = 记忆编码的最佳时刻
    重要信息应放在事件边界处 → 记忆效果最好
    事件边界由 Magliano (2011) 的动作分割研究定义

  画面复杂度与记忆:
    简单画面 + 长时长 = 高记忆保持
    复杂画面 + 短时长 = 低记忆保持 (信息过载)
    简单画面 + 短时长 = 中等记忆 (足够处理)
    复杂画面 + 长时长 = 中等记忆 (有时间处理但信息量大)

分镜应用:
  关键叙事信息 → 放在事件边界处 (动作完成的瞬间)
  线索植入 → 放在事件内部 (不在边界上 → 记忆编码弱 → Fair-play但不显眼)
  场景转换时认知成本最高 → 转场后第一个镜头应简单 (低I+S负荷)
  复杂信息 → 增加镜头时长 (给予K+E处理时间)
```

### Source 8: Magliano et al. (2011) — 事件分割与动作连贯

```
来源: Magliano, J. P., Miller, J., & Zwaan, R. A. (2001).
      "Indexing space and time in film comprehension."
      Applied Cognitive Psychology, 15(5), 533-545.

      及 Magliano, J. P., & Zacks, J. M. (2011).
      "The impact of continuity editing in narrative film
      on event segmentation." Cognitive Science, 35(8), 1489-1517.

核心发现:
  观众在观看电影时自动将连续的体验分割为离散的"事件":
    分割主要由动作变化驱动 (> 空间变化 > 时间变化)
    连续性剪辑 (continuity editing) 抑制事件边界感知
    非连续性剪辑触发事件边界

  跨文化验证:
    土耳其无电视暴露村庄的居民无法理解正反打 =
    "两个人在不同时空"而非"两个人在对话"
    → 连续性剪辑是习得的视觉语法，不是天生的

  事件分割层级:
    粗粒度事件 (场景级): 由空间/时间/角色大变化触发
    细粒度事件 (动作级): 由具体动作的完成触发
    编辑可以与这两个层级对齐或不对齐

分镜应用:
  (已在 shot-logic.md 法则 1 中形式化)
  扩展应用:
    粗粒度事件边界 → 场景转换点 → 适合使用建立序列
    细粒度事件边界 → 动作完成点 → 自然切点
    违反事件期望 → 制造惊奇 (但增加认知负荷)
```

### Source 10: Berlyne (1971) — 美学与激发理论

```
来源: Berlyne, D. E. (1971). Aesthetics and Psychobiology.
      Appleton-Century-Crofts.

核心发现:
  刺激的"唤醒电位" (arousal potential) 决定美学偏好:
    倒U型曲线: 中等复杂度/新奇度 = 最高偏好
    过低 → 无聊
    过高 → 不安/厌恶

  四种影响唤醒电位的属性:
    1. 新奇性 (Novelty): 与先前经验的差异
    2. 复杂性 (Complexity): 元素数量和关系
    3. 不确定性 (Uncertainty): 可预测程度
    4. 冲突性 (Conflict): 矛盾元素的共存

分镜应用:
  每个镜头的"新奇度"应在倒U型最优区间:
    完全重复上一个镜头 → 太低（无聊）
    完全无关联的画面 → 太高（混乱）
    继承一些元素 + 引入一些新元素 → 最优

  镜头序列的复杂度管理:
    简单-简单-简单 → 观众走神
    复杂-复杂-复杂 → 观众疲劳
    简单-中等-复杂-简单(呼吸)-中等-复杂 → 最佳参与度
```

## 二、注意力聚光灯规则 (Attentional Spotlight Rules)

### 规则集: 新镜头中观众的第一注视点

观众看到新镜头后，注意力在 100-300ms 内锁定到第一个注视点。
这个过程主要由外源性注意力驱动 (Posner, 1980)，不受意志控制。

```
优先级列表 (Priority Stack):
按优先级从高到低，第一个满足的条件决定第一注视点。

  P1: 面部 (尤其是眼睛)
      IF 画面中有人脸
      THEN 第一注视点 = 人脸的眼睛区域
      依据: Smith & Henderson (2008) 面部是最强的注意力锚点
      条件: 面部需占画面 > 5% 面积才触发
      例外: 多张面部 → 注视画面中心最近的面部

  P2: 运动中的物体 (画面内最大运动量)
      IF 画面中有运动元素且无面部 (或面部不显著)
      THEN 第一注视点 = 最大运动量的物体
      依据: Posner (1980) 外源性注意力被运动自动触发
      量化: 运动量 = 位移速度 x 物体面积
      例外: 多个运动物体 → 最大运动量的优先

  P3: 高对比区域 (亮度/色彩突变)
      IF 画面中无面部、无运动
      THEN 第一注视点 = 最高局部对比度的区域
      依据: Posner (1980) 外源性注意力被对比度触发
      实践: 光线打在脸上/物体上 = 人工制造对比度焦点

  P4: 屏幕中心偏好 (Center Bias)
      IF 画面中无面部、无运动、无明显对比区域
      THEN 第一注视点 = 画面中心区域 (中心 25% 面积)
      依据: Smith & Henderson (2008) 观众在无其他线索时
            默认注视屏幕中心
      原因: 电影院/屏幕的物理布局 + 主要内容通常在中心

  P5: 前一镜头注视点的延续位置
      IF 以上四条都不突出
      THEN 注视点 = 前一镜头结束时注视点在新画面中的对应位置
      依据: 注意力惯性，视线不会在切的瞬间跳跃
      实践: 匹配剪辑利用这个原理 → 两个镜头中的焦点在同一屏幕位置
```

### 规则集: 注视点之后的扫描路径 (Scan Path)

第一注视点确定后，观众的内源性注意力接管，按以下模式扫描：

```
扫描路径规则:

  RULE SCAN-1: 面部→身体→环境
      IF 从面部CU开始
      THEN 扫描: 眼睛 → 嘴 → 表情整体 → (如可见)手部 → 环境
      时间: 每个区域停留 200-400ms

  RULE SCAN-2: 主体→上下文
      IF 画面有明确主体 (人/物)
      THEN 扫描: 主体 → 主体周围环境 → 画面边缘
      时间: 主体 60-70% 时间，环境 30-40%

  RULE SCAN-3: 前景→背景 (深度扫描)
      IF 画面有明确的深度分层 (FG/MG/BG)
      THEN 扫描: 焦点层 → 次焦层 → 最远层
      时间: 焦点层 50-60%，次焦层 25-30%，最远层 10-20%

  RULE SCAN-4: 引导线跟随
      IF 画面中有明确的引导线 (道路、手指、视线方向)
      THEN 注视沿引导线方向移动
      依据: 内源性注意力被几何结构引导

  RULE SCAN-5: 文字优先
      IF 画面中有可读文字 (标牌、屏幕、笔记)
      THEN 注视点立即跳到文字并尝试阅读
      依据: 识字者无法抑制阅读冲动 (自动化过程)
      分镜应用: 画面中的文字总是会被读到
      → 线索植入/误导都可以通过可见文字实现
```

### 注意力聚光灯的分镜决策树

```
设计一个新镜头时的决策:

Q1: 这个镜头中，我最需要观众首先看到什么？
    → 将该元素放在最高优先级位置

Q2: 当前优先级配置是否符合叙事意图？
    IF 需要观众先看到人脸 (情感镜头)
    THEN 确保面部是最大/最清晰的元素 (P1 触发)

    IF 需要观众先看到物体 (线索镜头)
    THEN 确保面部不在画面中或不显著 (禁止 P1)
         + 物体在运动或有高对比 (触发 P2/P3)

    IF 需要观众扫描整个画面 (建立镜头)
    THEN 降低所有优先级 → 均匀分布元素
         + 增加镜头时长 (给予扫描时间)

    IF 需要隐藏信息 (Fair-play 线索隐藏)
    THEN 将线索放在低优先级位置:
         画面边缘、背景、低对比区域
         + 在高优先级位置放置分散注意力的元素
         依据: 注意力瞬脱 (attentional blink)
               观众在注视焦点时对非焦点区域的变化检测降低

Q3: 镜头持续足够长让观众完成必要的扫描吗？
    IF 信息在P1位置 → 1-2s 足够
    IF 信息需要扫描到P3/P4位置 → 至少3-4s
    IF 需要观众完成完整扫描 → 至少5-8s
```

## 三、认知负荷限制 (Cognitive Load Limits)

### 每景别的最大信息单元数

基于 Sweller (1988) 认知负荷理论 + Miller (1956) 工作记忆容量，
结合景别决定的视觉信息密度:

```
信息单元定义:
  一个"信息单元"= 观众需要有意识处理的一条叙事信息
  包括: 角色身份、角色情绪、物体存在、空间关系、
        动作含义、对白内容、声音信号、时间线索

按景别的最大信息单元 (Max Info Units per Shot):

  ECU (极近特写): 1-2 个信息单元
    画面: 单一物体/面部局部
    示例: "钢笔没墨了" (1个) 或 "钢笔没墨了 + 这不是她的笔" (2个)
    IF info_units > 2 in ECU
    THEN OVERLOAD → 观众无法同时注意微表情和两件以上事物
    修复: 拆分为两个 ECU 或改用 CU

  CU (特写): 2-3 个信息单元
    画面: 面部/上胸
    示例: "角色在微笑" + "笑容不达眼底" + "嘴角有抽动" (3个)
    IF info_units > 3 in CU
    THEN OVERLOAD → 面部表演的信息已经很密集
    修复: 延长时长 (给予处理时间) 或拆分

  MCU (中近景): 2-4 个信息单元
    画面: 头到胸
    示例: "角色表情" + "手部动作" + "服装细节" + "持有物品"
    IF info_units > 4 in MCU
    THEN WARNING → 接近上限但可能可行 (如果时长足够)

  MS (中景): 3-5 个信息单元
    画面: 腰以上
    示例: 角色身份 + 姿态 + 持有物 + 与他人空间关系 + 表情大意
    IF info_units > 5 in MS
    THEN OVERLOAD → 观众开始选择性忽略信息
    修复: 将不关键的信息移到后续镜头

  FS (全景): 3-5 个信息单元
    画面: 完整人物
    示例: 角色身份 + 全身姿态 + 环境概况 + 运动方向 + 另一角色
    IF info_units > 5 in FS
    THEN WARNING → 需要较长时长

  MLS/LS (远景): 5-8 个信息单元
    画面: 人物 + 大量环境
    示例: 多个角色 + 空间布局 + 环境细节 + 时间线索 + 氛围
    IF info_units > 8 in LS
    THEN OVERLOAD → 超过工作记忆容量
    修复: 增加镜头时长至 5-8s 或分阶段用运动引导注意力

  ELS (极远景): 环境级 (ambient)
    画面: 环境为主
    叙事信息应限制在 1-3 个 (地点、时间、氛围)
    视觉细节丰富但叙事信息应简单
    IF explicit_narrative_info > 3 in ELS
    THEN WARNING → 观众无法同时消化环境和复杂叙事信息
```

### 信息负荷与时长的关系

```
最低呈现时长公式 (Minimum Duration):

  min_duration(seconds) = info_units x 0.8 + base_duration(shot_size)

  其中 base_duration:
    ECU: 0.5s    CU: 1.0s     MCU: 1.5s
    MS: 1.5s     FS: 2.0s     LS: 2.5s     ELS: 3.0s

  示例:
    CU 有 3 个信息单元 → min = 3 x 0.8 + 1.0 = 3.4s
    MS 有 5 个信息单元 → min = 5 x 0.8 + 1.5 = 5.5s
    ECU 有 1 个信息单元 → min = 1 x 0.8 + 0.5 = 1.3s

  IF actual_duration < min_duration
  THEN RISK: 观众未能完全处理信息
    悬疑利用: 故意的信息过载 → 观众"知道看到了什么但没抓住" → 不安
    修复 (正常场景): 增加时长或减少信息

规则 LOAD-1: 简单画面 + 长时长 = 高记忆保持 (Shimamura 2013)
规则 LOAD-2: 复杂画面 + 短时长 = 低记忆保持 (信息过载)
规则 LOAD-3: 关键信息 → 使用 LOAD-1 模式 (简单+长)
规则 LOAD-4: 隐藏线索 → 使用 LOAD-2 模式 (复杂+短)
```

### 序列级认知负荷管理

```
规则 SEQ-LOAD-1: 连续高负荷上限
  IF 连续 3+ 个镜头的 info_units 都在景别上限
  THEN INSERT 一个低负荷"呼吸"镜头 (ELS环境/空镜/静止CU)
  依据: Sweller (1988) 持续高外在负荷导致学习(理解)失败
        Berlyne (1971) 持续高复杂度导致厌恶

规则 SEQ-LOAD-2: 场景转换的认知缓冲
  IF 从场景A切到场景B (跨场景切)
  THEN 场景B的第一个镜头应是低信息密度的建立镜头
  依据: Shimamura (2013) 场景转换是最高认知成本的切
        需要 I + S + K + E 全部四层处理
  实践: 不要在场景转换后立即给高密度特写

规则 SEQ-LOAD-3: 高潮前的信息清理
  IF 即将进入叙事高潮 (高信息+高情感)
  THEN 高潮前 2-3 个镜头应逐步降低信息密度
  逻辑: 清空工作记忆 → 为高潮的关键信息腾出处理空间
  反面: 故意在高潮前堆信息 → 信息过载 = 混乱感
        (有些导演有意为之: Fincher 的犯罪现场信息轰炸)
```

## 四、情感启动规则 (Emotional Priming Rules)

### 启动时长阈值

基于 LeDoux (1996) 双通道模型 + 启动效应实验:

```
三级启动层次:

  Level 1 — 快通路启动 (Subliminal Priming)
    最短时长: 100-200ms (0.1-0.2s)
    机制: 杏仁核直接激活，绕过意识
    效果: 模糊的不安/警觉，观众无法说出原因
    分镜应用:
      闪现帧: 在正常镜头中插入 1-3 帧的异常画面
      阈下声音: 极低音量的异常声音
      IF 目标 = 制造无意识不安
      THEN 使用 0.1-0.2s 的闪现 + 恢复正常
      THEN 观众会感到"哪里不对"但无法指出
      适用: 恐怖/心理悬疑的氛围营造

  Level 2 — 基础启动 (Basic Priming)
    最短时长: 1.5-3s
    机制: 皮层加工建立初步情感基调
    效果: 观众有意识地感受到某种情绪状态
    分镜应用:
      IF 目标 = 基本的对比切 (平静→紧张，安全→危险)
      THEN 启动镜头至少 1.5s，理想 2-3s
      THEN 对比切可以产生有效的情感反差
      示例:
        CU 角色微笑 (2s，暖光) → 硬切到黑暗走廊 → 对比生效
        CU 角色微笑 (0.5s) → 硬切到黑暗走廊 → 对比减弱 (启动不足)

  Level 3 — 深度启动 (Deep Priming)
    最短时长: 5-10s (或跨多个镜头的序列)
    机制: 完整的情境图式建立 + 情感浸入
    效果: 观众完全沉浸在一种情感/认知状态中
    分镜应用:
      IF 目标 = 强烈的情感颠覆 (安全世界突然崩塌)
      THEN 需要 5-10s+ 的"安全"建立 (可跨多个镜头)
      THEN 颠覆的冲击力 ∝ 启动的深度和时长
      示例:
        连续 3 个镜头 (共 8s) 展示温暖的家庭场景
        → 硬切到犯罪现场
        → 对比冲击力极强 (因为深度启动完整)
```

### 对比切效果公式

```
对比切冲击力 = f(启动深度, 对比幅度, 切的突然性)

  启动深度 = 启动序列的总时长 x 情感一致性
    (所有启动镜头都传达同一情感 → 一致性高)
    (启动镜头情感混杂 → 一致性低 → 启动削弱)

  对比幅度 = |情感A的valence - 情感B的valence|
    安全→危险: 高对比
    安全→中性: 低对比
    紧张→更紧张: 无对比 (不是对比切)

  切的突然性:
    硬切 (CUT): 最高突然性 → 最高冲击
    溶解 (DISSOLVE): 低突然性 → 降低冲击
    淡出→淡入 (FADE): 最低突然性 → 几乎无冲击

规则 PRIME-1:
  IF 对比切的目标冲击力 = 高
  THEN 启动时长 >= 3s AND 对比幅度 >= 高 AND 切 = 硬切

规则 PRIME-2:
  IF 对比切的目标冲击力 = 中
  THEN 启动时长 >= 1.5s AND 对比幅度 >= 中

规则 PRIME-3:
  IF 启动时长 < 1.5s
  THEN 对比切效果削弱至 < 50%
  WARNING: 除非使用 Level 1 阈下启动 (0.1-0.2s 闪现)

规则 PRIME-4: 启动方向不可逆
  IF 已经启动了"危险"基调 (冷色、低光、紧张音乐)
  THEN 突然切到"安全"画面 → 观众不会立即感到安全
  杏仁核的威胁信号需要 2-5s 才能消退
  → "虚假安全" 的镜头在观众神经层面仍携带不安
  分镜利用: 安全→危险→安全 = 观众在"安全"阶段仍然紧张
```

### 情感持续时间的神经约束

```
规则 EMOTION-DECAY:
  情感激活后的衰减:
    惊吓反应: 峰值 0.5-1s，完全消退 3-5s
    恐惧/威胁: 峰值 1-3s，完全消退 10-20s
    悲伤/同情: 峰值 3-5s，完全消退 30-60s
    温暖/安全: 峰值 5-10s，完全消退 60s+

  分镜应用:
    IF 需要持续恐惧
    THEN 每 10-15s 补充一次威胁信号 (维持激活)
    IF 惊吓后需要第二次惊吓
    THEN 间隔至少 3s (否则第二次效果减弱 → 适应效应)
    IF 需要从悲伤转到行动
    THEN 过渡至少 5-8s (否则情感断裂 → 观众困惑)
```

## 五、注意力同步规则 (Attentional Synchrony Rules)

### 高同步条件 (观众注视点集中)

基于 Smith & Henderson (2008) 和 Hasson et al. (2008):

```
规则 SYNC-1: 面部同步
  IF 画面中有清晰的单一面部 (CU/MCU)
  THEN 观众注意力同步度 >= 85%
  THEN 所有观众注视同一区域 (面部)
  THEN 此时剪切 → 高概率"编辑盲" (观众不注意到切)

规则 SYNC-2: 突然运动同步
  IF 画面中出现突然运动 (之前静止的元素开始动)
  THEN 观众注意力在 100ms 内同步到运动源
  THEN 同步窗口持续 500ms-1s
  THEN 此窗口内剪切 → 编辑盲

规则 SYNC-3: 声音引导同步
  IF 声音线索指向画面特定区域 (角色说话、物体发声)
  THEN 观众注视点转向声源在画面中的位置
  THEN 声音可以在镜头内重新引导注意力
  分镜应用:
    J-Cut (声音先行) → 在新镜头出现前，
    声音已经引导观众"期待"某个方向 → 切到该方向 → 无缝

规则 SYNC-4: 视线追踪同步
  IF 角色A的视线明确指向画面中某方向
  THEN 观众的注视点跟随角色视线方向移动
  THEN 在跟随完成时剪切 → 自然切 (切到角色看到的东西)
```

### 低同步条件 (观众注视点分散)

```
规则 DESYNC-1: 宽景多元素
  IF 画面是 LS/ELS 且包含 3+ 个等重要视觉元素
  THEN 观众注意力同步度 <= 40%
  THEN 不同观众看不同的地方
  应用:
    线索隐藏 → 在低同步镜头中放置线索
    → 只有部分观众会注意到 → Fair-play (信息在画面中)
       但 不显眼 (大多数观众没有注视到)

规则 DESYNC-2: 静态均匀场景
  IF 画面无运动、无面部、亮度/对比均匀
  THEN 注意力同步度最低
  THEN 观众执行自由浏览 (free viewing)
  应用:
    不适合放置关键信息 (无法保证观众看到)
    适合作为"呼吸"镜头 (让观众自由探索)

规则 DESYNC-3: 深焦多层
  IF 画面使用深焦 (所有深度层都清晰)
  THEN 观众在不同深度层之间自由选择
  THEN 同步度降低
  应用:
    Orson Welles 式深焦 → 观众参与度高但注意力分散
    → 不适合需要精确注意力控制的悬疑关键时刻
    → 适合"氛围"和"空间理解"
```

### 同步-反同步策略矩阵

```
                  高同步 (观众注视集中)    低同步 (观众注视分散)
                  ────────────────────    ────────────────────
关键信息传递      最佳选择                  风险: 信息可能被错过
                  CU/面部 + 声音引导       需要多镜头冗余确认

线索隐藏          不适合: 观众注意力集中    最佳选择
(Fair-play)       线索会被注意到            线索在画面中但不被注视

隐形切 (流畅)     最佳选择: 编辑盲          不适合: 观众在自由浏览
                  切不会被感知              切会被注意到

冲击切 (打断)     可行但效果减弱            最佳选择: 从分散到集中
                  已经在高注意 → 冲击有限    的转变本身就是冲击

悬念维持          面部CU锁住观众            环境镜头释放压力
                  + 负空间制造期待           + 但也降低紧张度
```

## 六、Kuleshov 效应形式化

### Kuleshov 函数

```
原理 (Kuleshov, 1920s):
  同一张中性面部 + 不同的前后镜头 → 观众赋予面部不同的"情感"

形式化:
  Kuleshov(Face(neutral), Context(X)) → Perceived_Emotion(f(X))

  Face 可以是: 中性面部、微妙面部 (轻微情绪)
  Context 可以是: 前一个镜头 (更强) 或后一个镜头 (较弱)
  f(X) 是上下文到感知情绪的映射函数
```

### 10 个常见上下文的 Kuleshov 映射

```
Context → Perceived Emotion of Neutral Face

  C1: 食物 (一碗热汤)
      → 饥饿/欲望
      规则: Face(neutral) + Context(food) → Emotion(desire/hunger)

  C2: 棺材/死者
      → 悲伤/哀悼
      规则: Face(neutral) + Context(death) → Emotion(grief/sorrow)

  C3: 儿童/婴儿
      → 温柔/慈爱
      规则: Face(neutral) + Context(child) → Emotion(tenderness)

  C4: 美女/帅哥
      → 欲望/倾慕
      规则: Face(neutral) + Context(attractive_person) → Emotion(desire)

  C5: 刀/武器
      → 恐惧/威胁感知
      规则: Face(neutral) + Context(weapon) → Emotion(fear/menace)

  C6: 钱/财富
      → 贪婪/渴望
      规则: Face(neutral) + Context(wealth) → Emotion(greed/longing)

  C7: 空旷的房间/空间
      → 孤独/失落
      规则: Face(neutral) + Context(emptiness) → Emotion(loneliness)

  C8: 紧闭的门
      → 挫败/被拒绝
      规则: Face(neutral) + Context(closed_door) → Emotion(frustration)

  C9: 监控画面/屏幕
      → 窥视/控制欲 或 被监视感
      规则: Face(neutral) + Context(surveillance) → Emotion(voyeurism/paranoia)

  C10: 镜子中的自己
      → 自我审视/分裂/不确定
      规则: Face(neutral) + Context(mirror_self) → Emotion(self_doubt)

悬疑特化扩展:

  C11: 咨询笔记/档案
      → 侵犯/泄露/危机
      规则: Face(neutral) + Context(confidential_file) → Emotion(violation/crisis)

  C12: 帽衫/匿名衣物
      → 威胁/不安
      规则: Face(neutral) + Context(anonymous_clothing) → Emotion(threat)
```

### Kuleshov 效应的分镜规则

```
规则 KULESHOV-1: 三镜头模式最强
  Face(A) → Context(X) → Face(A)
  第二次面部获得最强的 Kuleshov 赋予
  因为观众在看第二次面部时已经有了上下文

规则 KULESHOV-2: 中性面部 > 明确面部
  越中性的面部 → Kuleshov 效应越强
  明确的情绪面部 → 效应减弱 (面部自身情绪覆盖上下文)
  分镜应用: 需要强 Kuleshov → 指示角色做"微妙"表演

规则 KULESHOV-3: 上下文的停留时间影响效应强度
  IF Context 镜头 < 1.5s
  THEN Kuleshov 效应减弱 (基础启动不足，LeDoux Level 2)
  IF Context 镜头 >= 3s
  THEN Kuleshov 效应最强 (深度启动)

规则 KULESHOV-4: 反应链中的层叠 Kuleshov
  事件 → Face(A) → Face(B) → Face(C)
  每个面部都被前面的镜头(包括其他面部)赋予情感
  A 被事件赋予
  B 被事件 + A的反应赋予
  C 被事件 + A的反应 + B的反应赋予
  → 反应链中越后面的面部承载越多的叙事信息

规则 KULESHOV-5: 悬疑中的歧义 Kuleshov
  IF Context 可以有多种解读 (如一封信 → 内容未知)
  THEN Face 的感知情绪也变得歧义
  → 观众投射自己的猜测 → 增加参与度
  分镜应用: 悬疑核心场景使用歧义上下文 + 中性面部
            → 让观众自己"推理"角色在想什么
```

## 七、1/f 时长分布规则

### Cutting 的发现及其含义

```
来源: Cutting, DeLong & Nothelfer (2010)

1/f 噪声 (pink noise) 的特征:
  功率谱密度 S(f) ∝ 1/f^α，其中 α ≈ 1.0

  白噪声 (α=0): 每个镜头时长完全独立 → 混乱
  粉噪声 (α=1): 短期变化 + 长期模式 → 最佳参与度
  棕噪声 (α=2): 时长变化缓慢 → 可预测/无聊

人类注意力的 1/f 波动:
  注意力本身按 1/f 模式波动 (Gilden, 2001)
  → 电影节奏与注意力节奏匹配时 → 最佳参与度
  → 匹配 = 观众"被带着走"而非"对抗节奏"
```

### 分镜节奏的 1/f 检验

```
规则 RHYTHM-1F-1: 局部变化
  IF 任何连续 3 个镜头的时长完全相同 (如 3s, 3s, 3s)
  THEN WARNING: 局部过于规律 → 接近棕噪声
  修复: 至少一个镜头时长变化 >= 30% (如 3s, 2s, 3.5s)
  例外: 脉搏型节奏模式 (故意均匀 → 用于特定效果)

规则 RHYTHM-1F-2: 全局模式
  IF 镜头时长序列的自相关在 lag 5-10 处仍显著
  THEN 存在长期模式 → 好 (1/f 特征)
  IF 自相关在 lag 1 后立即降至 0
  THEN 纯随机 → 坏 (白噪声)

规则 RHYTHM-1F-3: 嵌套循环
  成功的节奏设计包含嵌套的时间循环:
    微循环 (3-5 个镜头): 呼吸模式 (长-短-长 或 加速-停顿)
    中循环 (10-15 个镜头): 张力弧线 (建设→高潮→释放)
    宏循环 (整场景): 整体弧线
  这三层循环的叠加自然产生 1/f 模式

规则 RHYTHM-1F-4: 变化率约束
  相邻镜头的时长变化率:
    |duration[n+1] - duration[n]| / duration[n]
    均值应在 30-70% 范围
    < 30% → 变化太小 (棕噪声倾向)
    > 70% → 变化太大 (白噪声倾向)
    30-70% → 1/f 区间
```

## 八、注视预测模型 (Gaze Prediction Model)

### 按景别的注视路径预测

```
ECU 注视路径:
  入口点: 画面中心的最高对比区域
  扫描路径: 几乎无扫描 (整个画面就是焦点)
  停留点: = 入口点 (单焦点锁定)
  预测精度: ~95% (几乎所有观众看同一个点)

CU 注视路径 (面部):
  入口点: 眼睛 (双眼之间的鼻梁区域)
  扫描路径: 眼→嘴→眼 (三角扫描)
  停留点: 如果面部有表情变化 → 变化区域
         如果面部平静 → 回到眼睛
  预测精度: ~85%

CU 注视路径 (非面部):
  入口点: 物体的最高对比/最详细区域
  扫描路径: 功能区域扫描 (如手表 → 表盘→指针→品牌)
  停留点: 最有信息量的区域
  预测精度: ~75%

MCU 注视路径:
  入口点: 面部 (如有)，否则手部动作
  扫描路径: 面部 → 手部 → 持有物 → 回到面部
  停留点: 说话时→嘴部  聆听时→眼部
  预测精度: ~70%

MS 注视路径:
  入口点: 面部 (如有)
  扫描路径: 面部 → 姿态 → 环境信息 → 回到面部
  停留点: 取决于叙事焦点 (动作→身体 / 情感→面部)
  预测精度: ~60%

LS 注视路径:
  入口点: 画面中心 → 最大运动 → 人物 (按优先级)
  扫描路径: 焦点人物 → 环境扫描 (左→右或沿引导线)
  停留点: 人物 (如有)，否则最高视觉权重区域
  预测精度: ~40-50%

ELS 注视路径:
  入口点: 屏幕中心偏好 → 最高对比区域
  扫描路径: 自由浏览 (无固定模式)
  停留点: 任何突出元素 (如果有人物 → 人物)
  预测精度: ~25-35%
```

### 注视引导技术

```
技术 GAZE-1: 运动引导
  画面中只有一个运动元素 → 100% 注视锁定
  IF 需要引导注视到特定区域
  THEN 让该区域有运动 + 其他区域静止

技术 GAZE-2: 光线引导
  高对比的光斑 → 吸引注视
  IF 需要引导注视
  THEN 用光线照亮目标区域 + 暗化其他区域

技术 GAZE-3: 焦点引导
  浅景深 → 只有对焦区域清晰 → 注视锁定到对焦区域
  Rack Focus → 注视跟随焦点移动
  IF 需要在一个镜头内转移注视
  THEN 使用焦点转移 (rack focus)

技术 GAZE-4: 角色视线引导
  角色看向画面外某方向 → 观众视线跟随
  下一个镜头如果从该方向开始 → 视线匹配 = 流畅
  IF 需要为下一个镜头预设注视方向
  THEN 让角色的视线指向该方向

技术 GAZE-5: 匹配位置引导
  前一镜头的注视终点 = 后一镜头的关键元素位置
  → 观众在切的瞬间已经在"看"新镜头的焦点
  IF 需要无缝切
  THEN 将新镜头的焦点放在前一镜头的注视终点位置
  依据: P5 前一镜头注视点延续 (Attentional Spotlight P5)
```

## 九、知觉桥接规则 (Perceptual Bridging Rules)

### 跨切连贯性的优先级排序

基于 Magliano et al. (2011) + Liao et al. (2023):

```
观众理解两个镜头属于"同一场景/事件"的知觉线索，
按效力从强到弱排列:

  Bridge-1: 动作连贯 (Action Continuity) — 最强桥接
    镜头A中的动作在镜头B中继续
    效力: 最强 → 观众100%感知"同一事件"
    示例: 角色伸手(A) → 手握到门把(B)
    依据: Magliano (2011) 动作是事件分割的主导因素

  Bridge-2: 空间连贯 (Spatial Continuity) — 强桥接
    镜头A和B共享可识别的空间特征
    效力: 强 → 观众90%感知"同一空间"
    示例: A中可见的书架在B中仍可见
    条件: 轴线保持 (180度法则) 加强空间连贯

  Bridge-3: 声音连贯 (Audio Continuity) — 中强桥接
    声音跨切延续 (L-Cut, 环境音延续)
    效力: 中强 → 即使画面跳跃，声音统一空间
    示例: 对话声音不中断 (L-Cut跨切延续)
    特殊: J-Cut (声音先行) 也建立桥接 → 预桥接

  Bridge-4: 时间连贯 (Temporal Continuity) — 中等桥接
    两个镜头的时间连续 (没有跳跃)
    效力: 中等 → 如果其他线索也支持
    示例: A中角色开始说"我觉得——" B中继续"——有问题"

  Bridge-5: 因果连贯 (Causal Continuity) — 弱桥接
    镜头B是镜头A的因果结果
    效力: 弱于物理连贯但强于无连贯
    示例: A中角色扔杯子 → B中杯子碎在地上
    注意: 需要观众的推理 → 更高认知负荷

  Bridge-6: 主题连贯 (Thematic Continuity) — 最弱显性桥接
    镜头A和B通过主题/符号相关
    效力: 最弱的显性桥接 → 需要观众有文化知识
    示例: A中角色哭泣 → B中下雨 (情感映射)
    注意: 这是蒙太奇的基础 → 依赖更高层认知
```

### 桥接失败的诊断

```
规则 BRIDGE-FAIL-1: 无桥接切
  IF 镜头A和B之间无任何 Bridge 1-6
  THEN 观众感知"跳到了完全不同的时空"
  这可能是:
    故意: 场景转换 → 需要建立序列重新锚定
    失误: 应该补充至少一个桥接线索

规则 BRIDGE-FAIL-2: 矛盾桥接
  IF Bridge-1 说"同一动作" 但 Bridge-2 说"不同空间"
  THEN 观众困惑 → 认知负荷激增
  示例: 角色在A中伸手 + B中手的动作继续但背景完全不同
  修复: 确保所有桥接线索一致

规则 BRIDGE-FAIL-3: 最低桥接要求
  同一场景内的切: 至少需要 Bridge-1 或 Bridge-2
  跨场景的切: 至少需要 Bridge-3 或明确的"场景转换"标记
  跨时间线的切: 需要 Bridge-4 的明确否定 (字幕/淡出) + Bridge-5 或 6
```

## 十一、压力测试 (R029 Adversarial Cases)

### Case 1: ECU 中的 5 个信息单元 (过载测试)

```
场景: ECU 林小曼的眼睛，要求传达:
  1. 她在害怕
  2. 她在说谎 (瞳孔偏移)
  3. 她看了一眼时钟 (眼球运动方向)
  4. 她的睫毛有泪水
  5. 她认出了某人 (瞳孔扩大)

认知负荷预测:
  ECU 上限 = 1-2 个信息单元
  5 个 = 严重过载

  Shimamura I-S-K-E 分析:
    I: 可以处理 (单一视觉区域)
    S: 困难 (5 个微表情变化需要区分)
    K: 困难 (需要匹配 5 种微表情的含义)
    E: 不可能 (无法同时执行 5 种推理)

  预测结果:
    观众最多注意到 1-2 个信息
    按优先级: 恐惧 (最显著情绪) + 泪水 (最高视觉对比)
    其余 3 个信息 → 被遗漏或混淆

  修复方案:
    方案A: 拆分为 3 个镜头
      ECU-1: 恐惧 + 泪水 (2 个 → OK)
      ECU-2: 眼球移动看时钟 (1 个 → OK)
      ECU-3: 瞳孔扩大/认出 (1 个 → OK)

    方案B: 升级景别 + 增加时长
      CU 面部 (3-4 个单元允许) + 8-10s 时长
      → 给观众足够时间逐一处理

    方案C: 信息分层
      画面信息: 恐惧 + 泪水 (2 个)
      声音信息: 呼吸变化提示紧张 (1 个)
      后续镜头: 时钟 INSERT + 认出反应 (2 个)
```

### Case 2: 0.3s 闪现帧 (阈下知觉测试)

```
场景: 在正常对话序列中插入 0.3s (约 7 帧) 的异常画面
  画面内容: 帽衫人影站在走廊尽头

认知知觉预测:

  知觉阈值分析:
    0.3s = 300ms > 快通路阈值 (100-200ms) → 可被快通路处理
    0.3s = 300ms < 慢通路完整处理 (500ms+) → 不被完全意识到

  LeDoux 双通路预测:
    快通路: 杏仁核激活 → 模糊的威胁/不安感
    慢通路: 部分激活 → "好像看到了什么" 但无法确认

  Smith 注意力同步预测:
    如果在高同步时刻插入 (如对话CU之间):
      → 大部分观众不会意识到 (编辑盲)
      → 但快通路的不安感仍然存在
    如果在低同步时刻插入 (如ELS之后):
      → 部分观众可能意识到"闪了一下"
      → 不安感更强 (因为有部分意识参与)

  Berlyne 新奇性预测:
    0.3s 的异常画面 → 高新奇性但低意识处理
    → 不安的"唤醒"但不到"厌恶"
    → 恰好在倒U型最优区间的上缘

  分镜建议:
    适合心理悬疑/恐怖 → 制造无意识不安
    不适合推理悬疑 → 信息太模糊无法被用作线索
    IF 需要作为 Fair-play 线索
    THEN 时长应增加到 0.8-1.5s (可被意识到但容易被遗忘)
    IF 纯粹制造氛围
    THEN 0.3s 是有效的阈下工具
```

### Case 3: 故意的注意力误导 (魔术师逻辑)

```
场景: 需要在 LS 中放置一个关键线索 (衣帽钩上的灰色帽衫)
  同时确保首次观看时大多数观众不注意到

注意力误导策略 (基于认知规则):

  策略 1: 优先级压制
    将帽衫放在低优先级位置:
      不在面部附近 (P1 不触发)
      不动 (P2 不触发)
      低对比 (灰色帽衫在暗色走廊 → P3 不触发)
      不在画面中心 (P4 不触发)
    在高优先级位置放置分散元素:
      角色面部在画面中心 (锁定 P1)
      角色有动作 (挂大衣 → 锁定 P2)
    预测: ~85% 观众不会注意帽衫 (首次观看)

  策略 2: 认知负荷利用
    在帽衫出现的同一镜头中给予高信息密度:
      角色进门 + 台词 + 内心独白 (VO)
    → 工作记忆被占满 → 无认知资源分配给帽衫
    依据: Sweller (1988) 高外在负荷 → 挤出低优先级信息

  策略 3: 同步窗口利用
    在观众注意力高度同步到其他目标时展示帽衫:
      角色说话的CU (高同步到面部)
      → 切到 LS 走廊 (帽衫在背景)
      → 但在 LS 中角色立即有动作 (高同步到动作)
      → 帽衫在视觉外围但在画面中
    依据: Smith & Henderson (2008) 编辑盲 + 变化盲

  策略 4: 情感干扰
    在帽衫出现前用高情感信息占据杏仁核:
      前一个镜头是关于咨询内容泄露的强情绪讨论
      → LeDoux 快通路被占用处理威胁信息
      → 慢通路也被情感推理占用
      → 帽衫的"异常信号"被淹没
    依据: LeDoux (1996) 情感处理优先于中性信息处理

  Fair-play 检查:
    帽衫在画面中 → 信息已呈现 ✓
    观众有可能注意到 (如果特别留意) → Fair ✓
    大多数观众第一次不会注意 → Play ✓
    (二刷时会注意到 → "啊，那件帽衫一直在那里!")
```

## 十三、可测试预测清单

以下预测可以通过眼动追踪或观众反馈测试验证:

```
预测 P-01: Shot 1 (ELS咨询室) 中，首次注视点在画面中心/窗户区域
  验证: 眼动追踪第一个 fixation 的位置
  依据: P3 (高对比) + P4 (中心偏好)

预测 P-02: Shot 4 (CU "有人跟踪我") 后的 Shot 5 切，
           < 20% 观众意识到了切本身
  验证: 被试者报告是否注意到"画面变化"
  依据: SYNC-1 (面部CU的编辑盲)

预测 P-03: 如果 Shot 3 的沉默时长从 4s 缩短到 1s，
           Shot 4 的台词冲击力评分降低 30-50%
  验证: 观众冲击力量表 (1-10)
  依据: PRIME-2 (基础启动需要 >= 1.5s)

预测 P-04: Shot 6 的 INSERT 笔记本，如果文字可读，
           90%+ 观众的注视点在 200ms 内跳到文字
  验证: 眼动追踪 SCAN-5 规则
  依据: SCAN-5 (文字优先/不可抑制的阅读冲动)

预测 P-05: ch1 结尾的帽衫线索 (LS走廊，同时有角色动作+内心独白)，
           首次观看时 > 75% 观众不会注意到帽衫
  验证: 事后提问 "你注意到走廊里挂着什么?"
  依据: P1-P2 优先级压制 + SEQ-LOAD 认知负荷占用

预测 P-06: 如果 Shot 7 (ECU+SIL) 的沉默时长为 3-5s，
           观众的情绪评分 (紧张度) 显著高于 1-2s 版本
  验证: 实时情绪评分 (拨盘法)
  依据: Level 3 深度启动 + EMOTION-DECAY (恐惧峰值 1-3s)

预测 P-07: 整个 7-shot 序列的时长方差如果符合 1/f 模式，
           观众的总体参与度评分高于均匀时长版本
  验证: 注意力测量 + 事后参与度量表
  依据: RHYTHM-1F (Cutting 2010)
```

## 十四、VERIFY 门增强 (Cognitive Load Check)

在 shot-logic.md 的镜头逻辑检验清单中追加认知维度检查:

```
### Level 5: 认知检验 (来自 cognitive-perception.md)

□ 认知负荷检查
  每个镜头的信息单元数是否在景别限制内?
  IF info_units > max_for_shot_size
  THEN FLAG: 考虑拆分、增加时长、或升级景别

□ 最低时长检查
  每个镜头的时长 >= min_duration(info_units, shot_size)?
  IF actual_duration < min_duration
  THEN FLAG: 信息可能未被完全处理

□ 注意力引导检查
  关键叙事信息是否在注意力优先级 P1-P3 位置?
  隐藏线索是否在低优先级位置?
  IF 关键信息在 P4-P5 位置
  THEN FLAG: 信息可能被错过

□ 情感启动检查
  对比切前是否有足够的启动时长?
  IF contrast_cut AND prior_priming < 1.5s
  THEN FLAG: 对比效果可能不足

□ 桥接检查
  相邻镜头之间是否有至少一个桥接线索?
  IF no_bridge AND same_scene
  THEN FLAG: 观众可能感知到空间/时间断裂

□ 序列负荷检查
  连续 3+ 个高负荷镜头后是否有呼吸镜头?
  IF consecutive_high_load > 3
  THEN FLAG: 观众认知疲劳风险
```
