## 一、场面调度的定义

Mise-en-scène（法语：放置在场景中）包含了**进入摄影机视野的一切**：

```
场面调度 = 角色位置 + 角色运动 + 布景 + 道具 + 灯光 + 色彩 + 服装
```

它不是"怎么拍"的问题（那是摄影），而是"拍什么"的问题。

> **Kurosawa 原则**: 在移动摄影机之前，先让画面内部的元素移动。
> 如果画面内部的调度足够精彩，一个静止的中景就能讲完一个故事。

## 二、Katz A-I-L 调度模型

Steven Katz 在《Shot by Shot》中提出，所有两人场景的调度可以简化为三种基本模式：

### A型：顶点型（Apex）

```
平面图:
       ■ A
      ╱ ╲
    ╱     ╲
  ╱         ╲
CAM ─────── ■ B

两个角色面对面，摄影机在侧面（三角形顶点）
```

**特征**:
- 角色彼此面对面
- 摄影机可以看到两人的侧面
- 双人镜头（Two-shot）的标准布局

**情感含义**:
- 对峙、争论、审讯
- 两人之间有明确的冲突轴线
- 紧张度最高的调度模式

**覆盖方案**:
```
Master: 双人中景（从侧面）
A的单人: OTS-A（从B肩膀后拍A）
B的单人: OTS-B（从A肩膀后拍B）
反应: A或B的CU
INSERT: 桌上的证据/道具
```

**悬疑应用**:
- 审讯场景的标准调度
- 对质场景
- 角色间的心理博弈

### I型：直线型（Inline）

```
平面图:
  A ──→ B ──→ CAM
  或
  CAM ──→ A ──→ B

两个角色在同一轴线上，摄影机在轴线延伸方向
```

**特征**:
- 角色前后排列
- 一个角色的背面/侧面在前景
- 深度调度——利用景深

**情感含义**:
- 跟随、引导、追逐
- 保护（前面的人保护后面的人）
- 背叛（后面的人注视前面的人的背影）

**覆盖方案**:
```
深焦: 两人同时清晰（Citizen Kane式）
变焦: 焦点从前人转到后人（注意力转移）
跟踪: Steadicam跟随前面的人
```

**悬疑应用**:
- 跟踪场景（A跟踪B）
- 角色走在前面说谎，后面的人看着ta的背影怀疑
- 两人走进黑暗——who will turn first?

### L型：L形（L-shape）

```
平面图:
  A ──→
  │
  ↓
  B          CAM ↗

两个角色成90度角，摄影机在对角位置
```

**特征**:
- 一个角色面向另一个的侧面
- 不完全的对视
- 空间中有一个"开放"的角

**情感含义**:
- 日常对话、自然互动
- 不直接的沟通
- 一方回避另一方

**覆盖方案**:
```
双人: 从对角拍摄，两人的不同朝向都可见
A的单人: 正面或3/4侧面
B的单人: 侧面或3/4侧面
```

**悬疑应用**:
- 角色在厨房做饭（面向操作台）+ 另一个角色在旁边说话
- 自然的场景中隐藏不自然的对话内容
- 回避眼神接触 = 在隐瞒什么

### 调度类型选择决策

```
这个场景中两个角色的关系是？
├── 对抗/冲突 → A型（面对面，强化对峙）
├── 引导/跟随 → I型（前后，强化关系）
├── 日常/自然 → L型（自然角度，放松）
└── 复杂/多面 → A→L→I 的转变（关系在场景中变化）
```

## 三、多人场景调度

### 三人场景

```
三角调度:
     A
    ╱ ╲
   B ── C

摄影机策略:
  - 三人Master → 逐个分离
  - 谁说话拍谁 + 最重要的听者
  - 两人联盟 vs 一人孤立 → 通过景别表达

悬疑应用:
  嫌疑人A + 侦探B + 证人C
  B审问A时 → 偷偷观察C的反应
  三角关系中的视线流动 = 怀疑的流动
```

### 群戏调度

```
核心原则:
  1. 找到"权力中心"——谁在主导这个场景？
  2. 权力中心在画面中的位置 = 在故事中的位置
  3. 用角色的位置变化表达权力的转移

技法:
  - 重要角色在画面中心/力量点
  - 边缘人物在画面边缘
  - 角色走动 = 权力流动
  - 角色离开画面 = 退出/被排除
```

## 四、人际距离学（Proxemics）

Edward T. Hall 的四区间理论在分镜设计中的应用：

### 四距离区间

```
亲密区 (Intimate): 0 - 45cm
  ├── 爱人之间的对话
  ├── 威胁性的逼近
  ├── 耳语、秘密
  └── 镜头：CU / ECU

个人区 (Personal): 45cm - 120cm
  ├── 朋友之间的对话
  ├── 私密但非亲密的互动
  ├── 安全感内的亲近
  └── 镜头：MCU / MS

社交区 (Social): 120cm - 360cm
  ├── 正式对话、商务互动
  ├── 角色保持安全距离
  ├── 礼节性的交流
  └── 镜头：MS / MLS

公共区 (Public): 360cm+
  ├── 演讲、审判
  ├── 完全的距离感
  ├── 权力展示
  └── 镜头：LS / ELS
```

### 距离变化的叙事含义

```
距离缩短 (靠近):
  → 亲密度增加 / 威胁逼近 / 权力压迫
  → 景别应同步变窄（MS → CU）

距离增大 (远离):
  → 疏远 / 恐惧回避 / 解脱
  → 景别应同步变宽（CU → MS → LS）

一方靠近一方后退:
  → 追逐/逃避 / 纠缠 / 控制
  → 交替使用两人的不同景别

同步靠近:
  → 共识 / 和解 / 对决前的最后距离
  → 从两人镜头推到各自特写
```

### 距离"入侵"

```
当角色进入不应该进入的距离区时:
  社交区 → 亲密区（越过个人区）= 威胁/不适
  用镜头表达:
    - 被入侵者的CU（不安的表情）
    - 入侵者的LOW ANGLE（显得更有威胁性）
    - 两人之间的空间用前景虚焦遮挡（压缩感）

悬疑应用:
  审讯者从桌子对面走到嫌疑人身旁 → 距离从社交区直接到亲密区
  → 嫌疑人的心理防线被物理性地侵犯
```

## 五、深度分层

每个镜头的画面都应该有深度——至少两层，理想三层：

### 三层模型

```
┌────────────────────────────────────────┐
│  Background (BG): 环境、氛围、隐藏信息   │
│  ┌────────────────────────────────────┐ │
│  │  Midground (MG): 主要动作、角色      │ │
│  │  ┌────────────────────────────────┐ │ │
│  │  │  Foreground (FG): 框架、遮挡、近景 │ │ │
│  │  └────────────────────────────────┘ │ │
│  └────────────────────────────────────┘ │
└────────────────────────────────────────┘
```

### 各层的叙事功能

```
前景 (FG):
  - 提供画面深度和层次
  - 框架元素（门框、植物、家具边缘）
  - 遮挡制造悬念（看不完整=想看更多）
  - 虚焦前景暗示"通过某物观看"

中景 (MG):
  - 主要动作区域
  - 对话双方
  - 核心叙事内容
  - 通常是焦点所在

背景 (BG):
  - 环境信息（时间、地点、氛围）
  - 隐藏线索（Fair-play设计的关键层）
  - 其他角色的反应
  - 暗示性元素（象征物、异常元素）
```

### 焦点控制

```
深焦（全部清晰）:
  所有三层都清晰 → 观众自由选择看什么
  Orson Welles / Citizen Kane 的标志性技法
  优势: 信息丰富，观众参与度高
  劣势: 注意力分散，不易引导

浅焦（选择性对焦）:
  只有一层清晰 → 强制引导观众注意力
  优势: 精确控制观众看什么
  劣势: 信息受限

焦点转移（Rack Focus）:
  在一个镜头内从A点对焦到B点
  效果: 注意力的视觉化转移
  经典用法:
    前景角色说话（清晰）→ 焦点转移到背景角色（清晰）
    = "真正重要的不是说话的人，而是听话的人"

悬疑用法:
  前景对话（清晰）+ 背景有可疑人影（虚焦）
  → 焦点突然转到背景 → 人影已经消失
  = 观众看到了什么？还是没看到？
```

### 景深决策规则 (v2.0形式化)

> 景深不是"美学选择"——它是**信息控制工具**。
> 深焦=信息民主(观众自己选择焦点)，浅焦=信息独裁(导演强制焦点)。
> 这个选择必须与当前镜头的信息需求匹配。

```
景深决策树 (DoF Decision Rules):

IF info_units_required > 3 AND viewer_should_choose_focus = TRUE
  → 深焦 (f/8 - f/16)
  → 所有三层(FG/MG/BG)同时清晰
  → 适用: 建立场景、复杂空间关系、公平线索(线索在BG但观众可能不注意)
  → 关联隐喻: M18(DEPTH IS TIME) — 多层时间共存
  → 示例: 叶知秋的咨询室全景(椅子、书架、窗户都是信息)

IF info_units_required <= 2 AND attention_must_be_forced = TRUE
  → 浅焦 (f/1.4 - f/2.8)
  → 只有一层清晰，其余虚焦
  → 适用: 情感特写、强制注意力、私密对话
  → 关联隐喻: M25(TIGHT FRAMING) — 世界被压缩
  → 示例: 叶知秋的面部特写(只有她的表情重要)

IF focus_shift_is_narratively_significant = TRUE
  → 焦点转移 (Rack Focus)
  → 在镜头内从A点变焦到B点
  → 适用: 注意力的叙事性转移、发现时刻、"真正重要的是__"
  → 关联隐喻: M18(DEPTH IS TIME) — 从一个时间层到另一个
  → 示例: 对焦从说话的叶知秋(FG)转到背景柜子上的帽衫(BG)

IF subjective_dreamy_state = TRUE
  → 极浅焦 (f/1.4, 大光圈)
  → 焦平面极薄，焦外柔化
  → 适用: 回忆、梦境、解离状态、药物影响
  → 关联隐喻: M22(FOG IS UNCERTAINTY) — 视觉的不确定
  → 示例: 叶知秋的空白记忆闪回(极浅焦=记忆不可靠)

IF documentary_objective_state = TRUE
  → 全深焦 (f/8+, 广角)
  → 全画面锐利清晰
  → 适用: 客观记录、证据展示、犯罪现场、监控画面
  → 示例: 监控录像中的叶知秋(全深焦=不可否认的客观)

IF temporal_layers_required > 1
  → 深焦 + 每层有叙事信息
  → 对焦选择 = 叙事者认为哪个时间层更重要
  → 见 visual-metaphor.md M18(DEPTH IS TIME) 详细规则
```

### 景深转换的叙事含义

```
深焦 → 浅焦 (场景内):
  = 世界在缩小 / 角色的认知在收窄 / 从全局到局部
  转换速率: 通过逐步调整镜头光圈或交替不同景深镜头实现

浅焦 → 深焦 (场景内):
  = 世界在扩大 / 角色获得新的全局认知 / 从局部到全局
  转换时机: 通常对应叙事中的"恍然大悟"/"退一步看清全貌"

一致深焦 (全场景):
  = 公平/开放 — 观众可以自由探索画面
  Welles/Bazin 传统: 尊重观众的选择权

一致浅焦 (全场景):
  = 控制/引导 — 叙事者严格控制信息流
  悬疑传统: 叙事者决定你看什么
```

### Arnheim 视觉重量法则 (v2.0新增)

> 基于 Rudolf Arnheim (1974)《Art and Visual Perception》——
> 画面中的每个元素都有"视觉重量"(Visual Weight)，
> 重量决定了元素吸引注意力的程度。
> 理解视觉重量 = 理解观众会先看哪里。

```
视觉重量的七个决定因素 (按影响力排序):

1. 人脸 (最重)
   → 人脸永远是画面中最重的元素，尤其是眼睛
   → 即使人脸很小、在画面边缘、被部分遮挡，观众仍然会先看脸
   → 分镜含义: 如果你不想让观众看脸，就不要让脸出现

2. 隔离度 (Isolation)
   → 画面中孤立的元素比群集的元素更重
   → 在一群人中的一个独行者 > 群体中的任何一个
   → 关联隐喻: M02(ISOLATION IS EXPOSURE)
   → 分镜含义: 孤立放置线索物件 = 增加其视觉重量

3. 面积 (Size)
   → 更大的元素更重
   → 但大面积的简单元素 < 小面积的复杂元素(见#5)
   → 分镜含义: 景别选择直接控制元素的面积→重量

4. 色彩饱和度 (Saturation)
   → 高饱和色比低饱和色更重
   → 纯红色在灰色背景中 = 极重
   → 关联隐喻: M16(RED IS DANGER)
   → 分镜含义: 关键物件使用高饱和色 = 无需特写就能吸引注意

5. 细节复杂度 (Complexity)
   → 细节丰富的元素比简单的元素更重
   → 复杂纹理的衣服 > 纯色衣服
   → 分镜含义: 背景中的复杂物件可能不当夺取注意力

6. 画面位置 (Position)
   → 右上方 = 最重的位置 (西方阅读习惯: 视线终点)
   → 左下方 = 最轻的位置 (视线起点)
   → 画面上方 > 画面下方
   → 画面中心 = 特殊重量(Arnheim的"中心引力")
   → 关联隐喻: M26(LEFT-RIGHT IS PAST-FUTURE)

7. 运动 (Motion)
   → 运动中的元素比静止的元素更重
   → 即使是微小的运动(呼吸、手指敲击)也增加重量
   → 分镜含义: 静止画面中的唯一运动 = 所有注意力
```

### 视觉重量在分镜中的应用

```
线索植入(Fair-play设计):
  想让线索"在画面中但不被注意":
    → 降低其视觉重量:
      - 放在左下方(最轻位置)
      - 降低饱和度(与环境色一致)
      - 不要隔离(放在其他物件中)
      - 确保画面中有更重的元素(人脸)吸引注意

  想让线索"在揭示时被注意到":
    → 增加其视觉重量:
      - 放在力量点(三分法交叉点)
      - 增加饱和度或独特色彩
      - 隔离(周围留空)
      - 使用INSERT单独展示

注意力竞争:
  IF 画面中同时有两个高重量元素
    → 观众会在它们之间快速扫视 → 不稳定的注意力
    → 用于制造"选择焦虑"(悬疑中: 哪个才是重要的？)
    → 或使用浅焦强制选择一个

视觉平衡:
  画面的视觉重量应该"平衡"——
  除非你故意制造不平衡(不安感):
    左重右轻: 重心在过去(M26关联)
    右重左轻: 重心在未来
    上重下轻: 压迫感
    下重上轻: 不稳定感(要"倒"了)
```

## 六、色彩与光线叙事

### 色彩心理

```
色彩温度的叙事函数:
  暖色调（橙/金/红）→ 安全、亲密、温暖、生命
  冷色调（蓝/青/灰）→ 危险、疏离、死亡、孤独
  中性调 → 客观、日常、过渡

色彩变化 = 叙事变化:
  暖→冷 : 从安全进入危险
  冷→暖 : 从危险回到安全（或虚假安全）
  高饱和→低饱和 : 从活力到衰败
  单色→多色 : 从限制到解放
```

### 色温决策函数 T_color (v2.0形式化)

> 基于 Elliot & Maier (2014) 色彩心理学元分析:
> 红色增加生理唤醒(心率+3-5bpm)，蓝色降低唤醒，
> 色温的冷暖直接影响观众的威胁感知阈值。
> 基于 Zettl (2013) 应用媒体美学: 色温可以量化为开尔文色温标度。

```
T_color(threat_level) — 色温决策函数

输入: threat_level ∈ [0.0, 1.0]（叙事威胁等级）

  threat_level 来源:
    0.0 = 完全安全（家、爱人、日常舒适）
    0.2 = 日常中的微小不安（叶知秋发现百乐钢笔）
    0.3 = 中性正常（标准日间场景）
    0.5 = 明确的不安（林小曼讲述跟踪经历）
    0.6 = 逼近的危险（调查中发现异常）
    0.8 = 直接的恐惧（面对威胁）
    1.0 = 极端状态（死亡、崩溃、现实破碎）

输出: 色温参数组

  T(0.0) = {
    kelvin: 3200K (钨丝灯/蜡烛暖光),
    saturation: 0.7 (偏高饱和),
    palette: amber/gold/warm-orange,
    effect: "安全、亲密、温暖"
  }

  T(0.3) = {
    kelvin: 5600K (日光中性),
    saturation: 0.5 (正常饱和),
    palette: daylight-neutral,
    effect: "正常、客观、中性"
  }

  T(0.6) = {
    kelvin: 7500K (阴天冷光),
    saturation: 0.4 (略低饱和),
    palette: cool-blue/slate,
    effect: "不安、疏离、隐约的危险"
  }

  T(0.8) = {
    kelvin: 9000K (钢蓝冷光),
    saturation: 0.2 (低饱和),
    palette: steel-blue/grey,
    effect: "恐惧、死亡、冰冷"
  }

  T(1.0) = {
    kelvin: N/A (去饱和/单色),
    saturation: 0.0-0.1 (近乎无色),
    palette: monochrome/bleached,
    effect: "极端、超现实、现实破碎"
  }

中间值线性插值:
  T(0.45) = lerp(T(0.3), T(0.6), (0.45-0.3)/(0.6-0.3))
```

### 色温转变速率规则

```
色温变化不是瞬间的——变化速率本身是叙事工具:

渐变（Gradual）: 跨5-10个镜头缓慢变化
  效果: 不知不觉中进入危险（观众的温水煮蛙）
  适用: 建构性悬念（audience slowly realizes something is wrong）
  示例: 叶知秋的公寓场景从T(0.2)经过整场戏到T(0.5)

突变（Sudden）: 单个硬切中色温跳变
  效果: 震惊、认知中断、世界观改变
  适用: 揭示时刻、恐惧事件、时空跳跃
  示例: T(0.3)→硬切→T(0.8) = 发现鞋上泥点的瞬间

振荡（Oscillating）: 在冷暖之间快速交替
  效果: 不稳定、分裂、精神状态失常
  适用: 解离状态、幻觉、记忆碎片
  示例: 叶知秋的空白记忆闪回: T(0.2)↔T(0.7)快速交替

回暖（Return-to-Warm）: 从冷→暖，但暖色中有微妙异常
  效果: 虚假安全（观众知道这不对，但角色不知道）
  适用: 叙事操纵——让读者放松警惕后再次打击
  示例: 危机后回到"正常"的T(0.3)，但比之前有0.05的偏冷
```

### 色温-调度-隐喻联动规则

```
色温不是孤立使用的——必须与调度和隐喻联动:

IF 调度类型 = A型(对峙) AND narr_gravity >= 4
  THEN T_color(0.5-0.7) + 侧光(M07阴影)
  → 冷色对峙 + 半明半暗面部

IF 隐喻 = M04(镜像) AND 场景类型 = 自我对峙
  THEN T_color(0.6-0.8) + 色温在镜子两侧可以不同
  → 真人侧暖于镜像侧 = 镜像是"更冷"的真相

IF 隐喻 = M24(夜) AND 时间 = 深夜
  THEN T_color(0.5-0.8) + 单一实际光源
  → 色温由实际光源决定(手机屏幕=冷蓝, 台灯=暖黄)
  → 暖色光源+冷色环境 = 安全岛效应

IF 场景结构 = 从日常到异常
  THEN 渐变从T(0.2)到T(0.6+)
  → 不要在场景开头就使用冷色——让色温"追踪"叙事
```

### 光线模式

```
高调光（High Key）:
  均匀明亮，阴影最少
  效果: 安全、正常、日常
  悬疑用法: 表面的安全感（反衬即将到来的黑暗）

低调光（Low Key）:
  强烈对比，大面积阴影
  效果: 危险、神秘、隐藏
  悬疑用法: 默认的悬疑光线设计

侧光（Side Light）:
  光源在侧面，半脸明半脸暗
  效果: 双面性、分裂、秘密
  悬疑用法: 不可靠叙事者 / 隐藏另一面的角色
  → 关联隐喻: M07(SHADOW IS HIDDEN SELF)

逆光（Back Light）:
  光源在角色身后
  效果: 剪影、神秘、未知身份
  悬疑用法: 凶手/神秘人物的登场
  → 关联隐喻: M17(LIGHT SOURCE IS AUTHORITY)

底光（Under Light）:
  光源在下方
  效果: 恐怖、怪异、非自然
  悬疑用法: 恐怖氛围（手电筒/火焰/屏幕光）
```

### 光线模式决策矩阵

```
IF 场景功能 = 日常建立 AND threat_level < 0.3
  → 高调光 + T_color(0.2-0.3)
  注意: 即使在日常中也可加入微弱的阴影暗示

IF 场景功能 = 悬念/调查 AND threat_level 0.3-0.6
  → 低调光 + T_color(0.5-0.6)
  阴影面积 ≥ 40% 画面

IF 场景功能 = 角色二重性展示
  → 侧光 + T_color(0.4-0.6)
  光影比 = 角色"光面/暗面"的比例
  → 叙事中角色真实面暴露得越多，光面越大

IF 场景功能 = 神秘人物登场
  → 逆光 + T_color(0.6-0.8) + 浅景深
  → 身份信息为零 = 轮廓可见但面部不可辨识

IF 场景功能 = 恐怖/超自然
  → 底光 + T_color(0.7-0.9) + 极度不饱和
  使用频率: 极低（一部作品中最多2-3次，保持冲击力）
```

## 七、视觉隐喻与象征

> **完整的视觉隐喻词典请参见: visual-metaphor.md (28个编号条目M01-M28)**
> 本节保留空间/镜子隐喻的概述并增加与词典的交叉引用。

### 空间即叙事

```
封闭空间:
  - 小房间、电梯、车内 → 被困、无法逃脱
  - 墙壁渐渐"收紧"（通过镜头变化实现）
  - 窗户/门 = 潜在的逃生路线（有没有打开？）
  → 对应隐喻: M08(CAGE IS ENTRAPMENT), M25(TIGHT FRAMING IS CLAUSTROPHOBIA)

开放空间:
  - 空旷街道、田野 → 暴露、无处躲藏
  - 开阔 ≠ 安全（在开阔中更加脆弱）
  → 对应隐喻: M02(ISOLATION IS EXPOSURE)

垂直空间:
  - 楼梯上方 = 优势位置
  - 楼梯下方 = 劣势位置
  - 角色在楼梯上的位置变化 = 权力变化
  → 对应隐喻: M01(POWER IS UP), M09(STAIRCASE IS POWER SHIFT)

门与窗:
  - 门 = 决定（打开/关闭 = 选择/放弃）
  - 窗 = 真相（看到/看不到 = 知道/不知道）
  - 锁上的门 = 被阻止的真相
  - 被打破的窗 = 暴力揭示的真相
  → 对应隐喻: M05(DOOR IS CHOICE), M06(WINDOW IS TRUTH)

走廊/通道:
  - 收窄的透视线 = 不可逃避的命运
  - 只有一个方向可走 = 没有选择
  → 对应隐喻: M12(CORRIDOR IS INEVITABILITY)
```

### 镜子的特殊地位

```
在悬疑类型中，镜子是最强大的视觉工具:
→ 完整的镜像隐喻设计见 visual-metaphor.md M04(MIRROR IS DUALITY)

镜子的含义:
  - 自我审视 / 自我对峙
  - 双重人格（镜像 ≠ 自我）
  - 不可靠叙事的视觉暗示（镜中看到"不该看到的"）
  - 真相（"镜子不说谎"——但镜子也反转了一切）

镜子的分镜技巧:
  1. 角色对镜说话 → 实际上在和"另一个自我"对话
  2. 镜中出现第三个人 → 恐惧/发现
  3. 镜中的动作与现实不同步 → 现实被扭曲
  4. 镜子破碎 → 自我认知崩塌 (叠加M15: BROKEN OBJECT IS BROKEN RELATIONSHIP)
  5. 通过镜子拍摄对话 → 间接性/不真实感

在《镜中访客》中:
  全剧的最终揭示发生在镜子前——
  这个空间选择本身就是叙事设计

  → 完整的跨章节镜像隐喻弧线设计见
    visual-metaphor.md §六("与《镜中访客》的全局隐喻设计")

## 八、调度演变与权力转移

### 场景内调度变化的叙事含义

```
场景开始时的调度 ≠ 场景结束时的调度
→ 变化本身就是故事

具体变化:
  角色站起来 → 权力提升
  角色坐下来 → 权力下降/放弃
  角色走向窗户 → 逃避/思考/观察
  角色走向门 → 即将离开/即将行动
  角色背对另一个角色 → 拒绝/隐藏/脆弱
  两人距离缩短 → 亲密/对抗升级
  两人距离增大 → 疏远/恐惧/解脱
  角色从画面中心移到边缘 → 失去主导权
  角色从边缘移到中心 → 获得主导权
```

### 摄影机运动与调度的配合

```
角色向右走 + 摄影机向右跟 = 跟随（观众跟角色走）
角色向右走 + 摄影机向左移 = 对抗（观众和角色方向相反=不赞同）
角色静止 + 摄影机推近 = 揭示（摄影机替观众接近角色）
角色靠近 + 摄影机后拉 = 保持距离（观众与角色保持距离=不信任）
角色和摄影机同时静止 = 凝固（时间暂停的效果）
```

## 十、《镜中访客》第1章空间设计应用 (v2.0新增)

将本文档的形式化规则应用于具体场景:

### 场景A: 咨询室——林小曼讲述被跟踪

```
叙事参数:
  scene_function: 对话 → 揭示(林小曼说出"那句话")
  threat_level: 0.2 → 0.5 (渐变)
  power_balance: 叶知秋主导(咨询师权威)
  character_count: 2
  info_density: 中(林小曼的描述+叶知秋的反应+环境线索)
  narrative_gravity: 3→4(当"那句话"说出时升级到4)
  primary_metaphor: M17(LIGHT SOURCE IS AUTHORITY) — 叶知秋控制咨询室的灯光

决策输出:
  调度: L型(自然对话) → 渐变到A型(当话题变沉重)
  色温: T(0.2)开始(暖色咨询室) → T(0.5)("那句话"说出后)
        转换方式: 渐变(跨8-10个镜头缓慢变冷)
  景深: 中等(f/4) → 关键台词时变浅(f/2.8, 只对焦说话者)
  光线: 侧光(暖色窗光从一侧照入)
        → 场景推进中窗光渐弱(天色变暗 = 叙事性的T_color降低)
  隐喻: M17(叶知秋的台灯=她的权威)
        M14(林小曼离开后的空沙发 = 隐性伏笔)

  "空气冷了一秒":
    → 镜头: CU叶知秋 + STATIC + 3s
    → 色温: 在这1秒内突变 T(0.3)→T(0.5)
    → 景深: 突然变浅(背景虚焦 = 世界消失了)
    → 声音: SIL: 3s
    → 这是渐变中的一个突变节点 = Elliot & Maier的arousal spike
```

### 场景B: 公寓深夜——发现帽衫和泥点

```
叙事参数:
  scene_function: 独处/反思 + 揭示(鞋上的泥)
  threat_level: 0.3 → 0.4(低威胁，角色自己在合理化)
  power_balance: 叶知秋与自己(她不知道的内在冲突)
  character_count: 1
  info_density: 高(帽衫、百乐钢笔、笔记、泥点——多个线索同时在场)
  narrative_gravity: 2→3(泥点发现时)
  primary_metaphor: M07(SHADOW IS HIDDEN SELF) + M24(NIGHT IS SUBCONSCIOUS)

决策输出:
  调度: 单人调度(叶知秋在公寓中移动)
  色温: T(0.3)(台灯暖光=安全岛) + T(0.5)(台灯范围外=冷色暗部)
        → 双色温: 暖色光源+冷色环境 = M17安全岛效应
  景深: 深焦(f/8) — 观众需要看到公寓中所有的线索物件
        → 关键: 帽衫在BG层(深焦中可见但视觉重量低)
              灰色帽衫+灰色衣架 = 低饱和+不隔离 = 低视觉重量
              叶知秋的脸 = 高视觉重量 → 观众看脸不看帽衫
              → 公平线索设计: 帽衫"在画面中"但不被注意
  光线: 实景光(台灯+手机屏幕光)
        → 叶知秋只在灯光范围内活动 = 她只能"看到"自己知道的
        → 公寓暗处 = 她不知道的自己(M07阴影)
  隐喻:
    M07: 公寓的暗区 = 她的潜意识
    M24: 深夜 = 潜意识活动时间(她即将入睡→进入"空白")
    M14: (隐性) 她走过衣帽架时没有停留 = "空椅子"的变体——
         物件在那里但她选择不"看到"它

  泥点发现:
    → 镜头: INSERT鞋底(3s) → CU叶知秋面部(4s)
    → 色温: 保持T(0.3-0.4)(关键: 不要过度反应——她在合理化)
    → 景深: 鞋的INSERT用浅焦(强制注意泥点)
    → 声音: 微弱的环境音(钟表?) + 无对白
    → 关键: 色温不要突变——叶知秋没有意识到这是异常
            观众的不安来自"她不应该这么平静"
```

### 空间设计总结: 第1章的渐进原则

```
空间参数在第1章中的整体弧线:

色温: T(0.2) ──────→ T(0.5) ──→ T(0.3-0.4) ──→ T(0.4)
      (日常暖)       (那句话)    (回到公寓假安全)  (泥点)
      ↑ 不是线性的——有一次突变(那句话)然后回暖(虚假)再缓慢下降

景深: 深焦 ─→ 浅焦 ─→ 深焦 ─→ 浅焦
      (建立)  (情感)  (公寓线索) (泥点INSERT)
