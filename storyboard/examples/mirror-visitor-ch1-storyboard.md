## 场景信息

- **POV**: 叶知秋（第一人称，不可靠叙述者——自我欺骗型+记忆扭曲型）
- **地点**: 叶知秋的心理咨询室 → 叶知秋的公寓
- **时间**: 周一，下午 → 深夜 → 次日早晨
- **本章核心功能**: 建立世界 + 引入案件 + 埋设线索C01（空白时间x2） + 埋设红鲱鱼H01（周明哲）
- **信息分配策略**: 限知叙事（观众 = 叶知秋的认知），但文本中的「合理化」信号是给二次阅读者的 fair-play 线索
- **三段落结构**:
  - A段: 咨询室场景（叶知秋日常 → 林小曼来访 → 核心揭示）
  - B段: 过渡回家（空间转换 + 帽衫发现）
  - C段: 公寓独处（分析推理 → 再次失忆 → 鞋上泥点）

```
   │                         门 ←──│

   摄影机轴线: 叶-林连线 (南北轴)
   摄影机侧: 西侧 (始终不越轴)
```

**调度模式**: A型（面对面，隔桌对坐）
**距离**: 社交区 (~200cm)，全程不变——职业距离的维持本身就是叙事信息
**运动轨迹**:
- B04: 林小曼从门口→沙发（I型进入→A型落座）
- B05-B10: 无大幅移动（A型维持 = 咨询的结构化空间）
- B11: 林小曼离开（画面外），叶知秋独坐（A型变为孤独的单人 = 对面空了）


**调度模式**: I型（角色进入空间）

## Step 7: ANNOTATE (注释)

### 关键镜头导演注释

**S01 — 开场**
- **意图**: 与S34形成首尾视觉呼应。叶知秋坐在床边的姿态、光线角度、构图位置应尽量一致——二次观看时，重复感会从「日常」变为「恐怖」。
- **参考**: 《消失的爱人》(Gone Girl, 2014) 开场——看似正常的日常隐藏深层异常。Fincher 用冷色调和精确对称的构图制造不安。
- **替代方案**: 可以从天花板俯拍(BIRD)开始——叶知秋躺在床上，仰视天花板=被困的暗示。但第一人称POV感会减弱。

**S13 — 核心揭示 (全章最重要的镜头)**
- **意图**: 这是全章的信息核心。林小曼复述"你妈妈说得对，你确实不值得被爱"时，镜头从CU极缓推至ECU。推近速度约2s/10cm。观众被不可抗拒地拉入她的恐惧——同时这句话的内容在叙事潜文本通道(N-channel)指向叶知秋自己。
- **参考**: 《沉默的羔羊》(Silence of the Lambs, 1991) Clarice 与 Lecter 对话中，Lecter 说出 Clarice 的创伤时，Demme 用 CU→ECU 的渐进推近。
- **替代方案**: 可以不拍林小曼而拍叶知秋的反应(延迟揭示 Idiom 3C)——先给反应再给原因。但第一章需要建立叶知秋作为「可靠叙述者」的印象，直接揭示更符合限知叙事。

**S14 — 叶知秋的震动**
- **意图**: 3秒完全无声的ECU。瞳孔微张+眼球微颤是不可控的生理反应——职业面具下的真实恐惧。这个镜头同时在三通道达到峰值: V(微表情), A(沉默的冲击), N(职业危机+可能指向自己)。
- **参考**: 《十二宫》(Zodiac, 2007) Fincher 在关键信息呈现时使用极端静止+沉默，让观众自己处理信息。
- **技术注意**: ECU需要演员极精确的微表情控制。瞳孔变化可能需要后期轻微调整。

**S23 — 独坐咨询室**
- **意图**: 全章的情感谷底+重压顶点。MLS缓拉——叶知秋在不断扩大的空间中显得越来越小。对面沙发的空缺=林小曼的缺席=叶知秋必须独自面对。窗外暮色=时间在她焦虑中流逝。
- **参考**: 《迷魂记》(Vertigo, 1958) Scottie 独坐场景——Hitchcock 用逐渐扩大的空间表达角色的心理收缩。

**S28 — 帽衫揭示**
- **意图**: Fair-play 设计的核心。帽衫在画面中停留5秒——足够让细心的观众注意到「灰色帽衫」=林小曼描述的跟踪者的穿着。但叶知秋只是「大概是吧」地略过。观众如果在此刻产生不安，在ch7就能提前推理出真相。
- **参考**: 《消失的爱人》——Amy 日记中的物品特写，每个都是后续揭示的伏笔。
- **替代方案**: 可以用Dolly Zoom拍帽衫——空间扭曲暗示认知失调。但ch1应维持叶知秋的「正常」表象，过于风格化会过早破坏不可靠叙述的伪装。

**S32 — 意识消散**
- **意图**: 这不是「角色睡着了」——这是解离状态的前兆。蒸汽消失=时间流逝的视觉隐喻。头缓缓低下=意识在主人格不知情的情况下「交出控制权」。极低频嗡鸣(barely audible)是唯一的声音暗示——二次观看时会意识到这是「另一个人格正在醒来」。
- **参考**: 《搏击俱乐部》(Fight Club, 1999) Jack 在办公桌前「失神」的镜头——单帧闪回被替换为渐进的意识模糊。

**S36 — 章末钩子**
- **意图**: 3秒完全无声+缓推近。泥点的湿润质感、棕色泥土的细节——这不是「白天踩到的」，这是夜间户外行走的证据。与S03的脚底灰形成升级关系（灰→泥=更明确的证据）。镜头在泥点上停留足够长，让观众「自己得出结论」而非被VO告知。
- **参考**: 《记忆碎片》(Memento, 2000) 每段开头的物证特写——物件自己说话，角色的理解永远比观众慢一步。
