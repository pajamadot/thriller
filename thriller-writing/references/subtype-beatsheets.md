# 子类型节拍表速查

> 每种悬疑子类型的关键节拍差异一览
> 基于 Truby 14种类型特定节拍 + 实战经验

---

## 本格推理（Whodunit）

```
节拍表（20章示例）：
  ch1  [I打开] 发现尸体/案件。读者："谁干的？"
  ch2  介绍 5-8 个嫌疑人（每人都有动机片段）
  ch3  第一轮审问——每人都有可疑之处
  ch4  发现第一条关键线索（但含义不清）
  ch5  [I打开] 次级谜团出现（受害者的秘密）
  ch6  第一个嫌疑人被排除（红鲱鱼消解）
  ch7  新的证据让嫌疑指向改变
  ch8  [E打开] 第二起事件（新线索或新威胁）
  ch9  中点翻转：最大假设被推翻
  ch10 [I关闭] 次级谜团解答——但答案引出新问题
  ch11 嫌疑范围缩小到 2-3 人
  ch12 [E关闭] 第二起事件解决
  ch13 关键线索的含义改变（重新解读）
  ch14 侦探被误导——走入错误方向
  ch15 最黑暗时刻：案件似乎无解
  ch16 最后一条线索到位
  ch17 [C打开] 侦探面对个人挑战
  ch18 集合所有人——侦探开始推理展示
  ch19 [C关闭] 侦探通过破案完成个人转变
  ch20 [I关闭] 真凶揭示 + 诡计机制完整呈现 + 余波
```

## 心理悬疑（Psychological Thriller）

```
节拍表（8章短篇示例，参照《镜中访客》）：
  ch1  建立叙述者的"可靠"形象。暗示裂缝（空白时间）。
  ch2  事件发生——叙述者开始"调查"。
       [实际上：读者正在被叙述者的框架锁定]
  ch3  调查深入——外部嫌疑人出现。
       [暗线：叙述者叙述中的微小矛盾开始累积]
  ch4  中点：外部嫌疑人被排除。"如果不是别人..."
       [信任弧线开始动摇]
  ch5  另一个角色开始质疑叙述者。
       叙述者自己的记忆出现裂缝。
  ch6  所有外部可能性被排除。
       叙述者独自面对不安。
  ch7  物理证据出现——无法否认。
       [叙述的可靠性崩塌]
  ch8  叙述者面对真相。
       [读者的认知框架被重构——回想前文，一切改变含义]

关键差异：真正的"调查"对象不是案件，而是叙述者自己。
```

## 社会派推理

```
关键节拍差异：
  - 犯罪动机在每一幕都要"升级"：个人→关系→制度→社会结构
  - 中点不是"谁干的"翻转，而是"为什么的深度"翻转
  - 揭示不仅是"谁是凶手"，更是"什么制度让这个人别无选择"
  - 每个嫌疑人代表社会的一个层面
  - 侦探的个人弧光与社会认知相关
```

## 惊悚悬疑

```
关键节拍差异：
  - 第一幕必须在 15% 前建立即时物理威胁
  - 每个场景都有"如果主角失败会怎样"的明确 stakes
  - Yes-but/No-and 节奏最密集——每 2-3 页一次反转
  - 中点是"猎人与猎物关系反转"
  - 第三幕的揭示可以在追逐/对抗中发生（不需要"集合推理"）
  - 对抗式悬念设计最适用：系统性摧毁主角的逃脱计划
```

## 密室推理

```
关键节拍差异：
  - 第一幕用 30% 的篇幅建立空间（含图表）
  - 空间的每个细节都可能是线索——读者和侦探同时"勘查"
  - 中点翻转通常是"密室不是我们以为的那种密室"
  - 揭示分两层：先"怎么做到的"（机制），再"为什么"（动机）
  - 参照 impossible-crime.md 的逆向工程法
```

## 倒叙推理（Howcatchem）

```
关键节拍差异：
  ch1-3  完整展示犯罪过程。读者看到一切。
  ch4    侦探登场。从零开始。读者知道答案但侦探不知道。
  ch5-6  侦探的初步调查——读者看着 ta 靠近或偏离真相。
         [悬念来自"侦探能不能发现"而非"谁是凶手"]
  ch7    侦探找到犯人计划中的第一个破绽。犯人开始紧张。
  ch8-9  猫鼠游戏——犯人试图补救，侦探步步逼近。
  ch10   侦探掌握决定性证据。犯人的完美计划崩塌。

张力来源完全不同：不是"谜团"而是"追逐"。
信息不对称反转：读者 > 侦探（而非经典的 侦探 > 读者）。
```

## 日常推理（Cozy Mystery）

```
关键节拍差异：
  - 前 20% 建立社区和日常（烘焙/书店/猫咖...这是类型承诺）
  - 犯罪"发生在幕后"——不直接描写暴力
  - 调查 = 社区八卦 + 偶然发现 + 业余推理
  - 嫌疑人都是邻居/朋友——增加个人 stakes
  - 中点可以是"社区秘密被揭开"而非传统的调查翻转
  - 揭示后社区关系的修复比正义实现更重要
  - 节奏更慢，角色互动占 60%，推理占 40%
```
