# 不可能犯罪设计

> 密室杀人与不可能犯罪的构造方法论
> 参考 John Dickson Carr "密室讲义"、岛田庄司空间诡计、绫辻行人建筑推理

---

## 一、Carr 密室分类（现代适配版）

### 机械类（物理手段制造密室）

| 类型 | 机制 | 示例 |
|------|------|------|
| 门锁操控 | 从外部操控内部锁具 | 线/磁铁/冰块操控门栓 |
| 窗户操控 | 从外部密封窗户 | 自动关闭装置/远程操控 |
| 自动装置 | 定时器/机关触发犯罪 | 冰块融化释放毒气、定时炸弹 |
| 隐藏通道 | 不可见的出入口 | 暗门、通风管道、地下通道 |
| 错觉空间 | 房间的实际结构与看起来不同 | 双层墙壁、移动隔板 |

### 心理类（认知操控制造"不可能"）

| 类型 | 机制 | 示例 |
|------|------|------|
| 时间错觉 | 犯罪时间与被认为的不同 | 尸体被提前/延后发现 |
| 身份替换 | 被认为在场的人其实不在 | 替身、变装、录音 |
| 死因误判 | 实际死因与表面不同 | 看似被杀实为自杀/意外/早已死亡 |
| 空间误判 | 犯罪地点与被认为的不同 | 尸体被移动、犯罪发生在其他房间 |
| 目击者操控 | 目击者看到的不是真相 | 距离/角度/光线造成误判 |

### 组合类（最高难度）

```
机械手段 + 心理操控的组合：

例：一个"密室"实际上从未真正被锁上（心理），
    但犯人用机械装置制造了"锁门声"（机械），
    加上目击者在错误时间看到"死者还活着"（时间错觉）。

→ 三层诡计叠加 = 不可能感最强
→ 但揭示时需要逐层剥离，每层都要有独立的线索支持
```

## 二、密室设计的逆向工程法

```
步骤 1：确定解法
  "凶手实际上是怎么做到的？"
  → 解法必须在物理/逻辑上完全成立

步骤 2：确定"不可能"的表象
  "读者看到的'不可能'是什么？"
  → 表象必须看起来确实不可能

步骤 3：确定哪些条件制造了表象
  "什么条件让'不可能'看起来成立？"
  → 这些条件中，至少一个是假的/被误解的

步骤 4：设计线索指向假条件
  "读者怎么才能发现那个假条件？"
  → 线索必须公平呈现

步骤 5：确保解法满足"啊哈"而非"怎么可能"
  "揭示后读者的反应是什么？"
  → 好的密室解法让人说"原来如此！精妙！"
  → 差的密室解法让人说"这也行？太勉强了"
```

## 三、空间作为诡计（绫辻行人方法）

### 建筑即角色

```
将建筑/空间视为故事中的一个"角色"：

  建筑的"性格"：它的设计原则、怪癖、历史
  建筑的"秘密"：隐藏的房间、不为人知的结构
  建筑的"谎言"：看起来是这样但实际上不是

设计清单：
  □ 绘制精确的平面图（给读者看）
  □ 在平面图中藏一个不显眼的异常
  □ 让角色在对话中讨论建筑（合理化介绍空间）
  □ 至少一个角色对建筑有超出正常的了解
  □ 建筑的某个特征在揭示后改变含义
```

### 图表契约（新本格传统）

```
向读者提供：
  - 人物关系图（章节前）
  - 建筑平面图（密室场景前）
  - 时间线图表（复杂时间诡计时）

这是与读者的公平性契约：
  "所有解谜需要的空间信息都在图中。"

设计要点：
  图表必须精确——但精确不等于完整。
  可以在图中省略一个小细节（如一扇不起眼的通风口），
  只要这个省略在文字叙述中有所提及。
```

## 四、不可能犯罪的测试清单

```
物理可行性测试：
  □ 在现实中能做到吗？（或在故事世界的规则下能做到吗？）
  □ 需要的时间是否足够？
  □ 需要的技能/工具是否合理？
  □ 是否有不可控的风险因素？（如果犯罪需要完美运气才能成功→太勉强）

逻辑完整性测试：
  □ 所有"不可能"的条件是否都有解释？
  □ 解释之间是否自洽？
  □ 揭示后回看，是否所有细节都对得上？

公平性测试：
  □ 解法所需的关键知识，读者是否都被告知了？
  □ 关键线索是否在密室发现时就已呈现？
  □ 是否依赖读者不可能知道的专业知识？

"啊哈"测试：
  □ 解法是否巧妙（而非仅仅合理）？
  □ 揭示是否让人想说"为什么我没想到"？
  □ 解法是否与故事主题有关联（而非纯粹的机械解谜）？
```
