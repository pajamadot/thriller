# /check 评分标准详细定义

> 每个维度每个分数段的具体标准，消除评分主观性

---

## 悬念张力

| 分段 | 标准 |
|------|------|
| 1-3 | 场景开始和结束时读者的好奇心没有变化。无钩子。无信息不对称。 |
| 4-6 | 有章末钩子但力度不足。有一些悬念但可预测。读者可能在此处放下书。 |
| 7-8 | 至少一个有效的信息不对称。章末钩子让人想翻页。场景内有期望→障碍→转折的弧线。 |
| 9-10 | 多层悬念同时运行。戏剧反讽有效利用。读者不可能在此处放下书。Yes-but/No-and 节奏精准。 |

## 线索技术

| 分段 | 标准 |
|------|------|
| 1-3 | 线索缺失或过于明显。红鲱鱼不可信。公平性严重不足。 |
| 4-6 | 线索存在但隐藏方式单一（都是物证或都是对话）。红鲱鱼有但说服力不够。 |
| 7-8 | 线索用功能遮蔽/情感遮蔽自然隐藏。红鲱鱼有独立叙事价值。通过双读者测试。 |
| 9-10 | 线索在首次阅读中完全隐形，二次阅读时显而易见。红鲱鱼的消解本身推进了故事。认知偏差被巧妙利用。 |

## 人物一致性

| 分段 | 标准 |
|------|------|
| 1-3 | 角色行为为情节服务而非性格驱动。对话缺乏辨识度。角色罗盘四维缺失。 |
| 4-6 | 角色在大部分时候自洽。对话有基本辨识度。但在压力场景下可能"出戏"。 |
| 7-8 | 角色行为始终由性格和知识状态驱动。对话风格独特。角色罗盘四维清晰。 |
| 9-10 | 角色在本章的表现既符合已建立性格又展现了新的面向。对话去掉名字仍能辨认说话人。伪盟友/盟友攻击节拍精准。 |

## 节奏控制

| 分段 | 标准 |
|------|------|
| 1-3 | 全章节奏单调。该快的地方慢，该慢的地方快。 |
| 4-6 | 有节奏变化但不够精准。某些段落可以更紧凑。 |
| 7-8 | 长短句交替有效。对话/描述比例符合场景类型。场景过渡流畅。 |
| 9-10 | 节奏变化与情绪完美匹配。关键时刻有"子弹时间"延伸效果。叙事速度五档（省略/概述/场景/延伸/停顿）运用自如。 |

## 逻辑严密性

| 分段 | 标准 |
|------|------|
| 1-3 | 明显的逻辑漏洞。时间线冲突。角色知道不应该知道的信息。 |
| 4-6 | 基本逻辑成立但有小疏漏。知识状态矩阵有 1-2 处不一致。 |
| 7-8 | 所有推理链条完整。知识状态矩阵完全一致。时间线无冲突。 |
| 9-10 | 逻辑不仅无漏洞，而且推理过程本身就是阅读乐趣。竞争叙事框架有效：侦探和犯人的叙事都有说服力。 |

## 文学质量

| 分段 | 标准 |
|------|------|
| 1-3 | 文字只传递信息，无风格。全是"他说""她做"。无感官描写。 |
| 4-6 | 偶有好句子但整体平庸。感官描写存在但不突出。无核心意象。 |
| 7-8 | 有辨识度的文风。至少 1 个有效的感官描写段落。核心意象出现。有 1 个可记住的句子。 |
| 9-10 | 每段都有文学质量。Chandler 式精准比喻或 Tana French 式内在化意象。核心意象贯穿且在揭示时升华。2+ 个"金句"。 |
