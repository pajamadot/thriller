# Thriller Writing Quick Reference Card

## Commands
```
/start → /theme → /trick → /characters → /structure → /scene N → /audit → /check N → /revise → /reveal → /export
```

## Sub-type Cheat Sheet

| Type | Workflow | Structure Focus | Key Technique |
|------|----------|----------------|---------------|
| Honkaku | Trick-first | Act 2 (clue collection 60%) | Fair-play clues, Knox check |
| Social | Char-first | Even distribution | Motivation escalation: personal→systemic |
| Psychological | Char-first | Act 3 (cognitive reversal) | Unreliable narrator, trust arc |
| Thriller | Structure-first | Act 2B (chase/escape) | Adversarial plan-failure |
| Locked-room | Trick-first | Act 1 (space) + Act 3 (mechanism) | Spatial puzzle, diagrams |
| Narrative trick | Trick-first | Act 3 (narrative subversion) | Dual-reading design |
| Procedural | Structure-first | Even, procedural realism | Legal/forensic accuracy |
| Inverted | Any | Act 1 (crime shown) + Act 2-3 (detection) | Cat-and-mouse tension |
| Cozy | Char-first | 60% community / 40% mystery | Off-screen crime, amateur sleuth |

## Per-Scene Checklist
```
□ At least one knowledge state changed
□ Yes-but or No-and outcome (never Yes-and or No-but)
□ Chapter-end hook (info/threat/cliff/twist/decision)
□ Clue operation logged in notes (plant/activate/reinterpret/dissolve)
□ Suspicion shift noted (who↑ who↓)
□ At least one relationship/power position changed
□ Reader's expected theory at this point
```

## Character Compass (per major character)
```
Lack:       What's missing? (wound)
Yearning:   What do they truly need? (unconscious)
Resistance: What prevents pursuit? (defense mechanism)
Desire:     What do they chase? (often wrong substitute)
```

## Ensemble Engine (per key character)
```
Mask:        What image do they actively perform?
True intent: What are they actually trying to secure?
Leverage:    What resource gives them power here?
Dependency:  Whose approval/protection/money/info do they need?
Secret:      What collapses if this gets exposed?
Rupture:     What event makes them switch sides immediately?
```

## Clue Density Guide
```
0-25%:  1-2 clues/chapter (hidden in world-building)
25-50%: 2-3 clues/chapter (investigation-driven)
50-75%: 1-2 clues/chapter (reinterpretation > new clues)
75-100%: 0-1 clues/chapter (full retrospection)
```

## Red Herring Lifecycle
```
Plant → Nurture → Strengthen → Deflect → Dissolve
(each stage serves story beyond just misleading)
```

## Information Ladder
```
Case surface → Suspect ecology → Relationship flip → Motive inversion → Emotional truth
Rule: answer 1 question, open 1-2 more expensive questions
```

## Knowledge State Quick Check
```
Who knows what right now?
  Author: 100%  |  Culprit: 90%  |  Detective: ?%  |  Reader: ?%
Every scene: at least one of these percentages changes.
```

## Fair-Play Dual Reader Test
```
Naive reader: Should NOT guess truth before reveal → surprise ✓
Detective reader: SHOULD be able to deduce from clues → fair play ✓
Both satisfied = fair mystery
```

## Dialogue Duel Checklist
```
□ Surface topic ≠ actual thing being fought over
□ Each speaker is probing, hiding, or buying time
□ At least one answer creates a bigger problem
□ Someone gains or loses leverage by the end
```
